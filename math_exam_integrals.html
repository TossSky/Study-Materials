§1. Площадь плоской фигуры (фундамент теории)
§1

Площадь плоской фигуры. Мера Жордана

Определение 1.1. Диаметр множества

Пусть A — ограниченное множество в R². Диаметр множества A:

d(A) = sup { ρ(a, b) : a, b ∈ A }

где ρ(a,b) — евклидово расстояние между точками a и b.

Замечание 1.1. Площадь многоугольной фигуры

Для многоугольной фигуры P площадь S(P) обладает свойствами:

  • Монотонность: если P₁ ⊂ P₂, то S(P₁) ≤ S(P₂)
  • Аддитивность: если P = P₁ ∪ P₂ и P₁ ∩ P₂ не содержит внутренних точек, то S(P) = S(P₁) + S(P₂)
  • Инвариантность: площадь не меняется при движении (параллельный перенос, поворот)

Определение 1.2. Внутренняя и внешняя площади

Пусть F — ограниченная фигура на плоскости.

Внутренняя площадь (вписанные многоугольники):

S*(F) = sup { S(P) : P ⊂ F, P — многоугольная фигура }

Внешняя площадь (описанные многоугольники):

S*(F) = inf { S(Q) : Q ⊃ F, Q — многоугольная фигура }

Всегда S*(F) ≤ S*(F).

Фигура F называется квадрируемой (измеримой по Жордану), если S*(F) = S*(F). В этом случае общее значение называется площадью S(F).

Теорема 1.1. Критерий квадрируемости

F квадрируема ⟺ ∀ε > 0 ∃P ⊂ F, Q ⊃ F (многоугольные): S(Q) − S(P) < ε

Определение 1.3. Множество меры ноль

Множество F имеет меру ноль (по Жордану), если:

∀ε > 0 ∃ Q ⊃ F (многоугольная фигура): S(Q) < ε

Иными словами, F можно покрыть многоугольной фигурой сколь угодно малой площади.

Следствие к Теореме 1.1

F квадрируема ⟺ граница ∂F имеет меру ноль
Это ключевой результат! На экзамене часто спрашивают: «Когда фигура квадрируема?» — ответ: когда её граница имеет меру ноль.

Лемма 1.1. Кривая имеет меру ноль

Если f(x) непрерывна на [a,b], то график y = f(x) имеет меру ноль.

Доказательство (через теорему Кантора):

  1. f непрерывна на [a,b] (компакт) → по теореме Кантора f равномерно непрерывна
  2. ∀ε > 0 ∃δ > 0: |x₁ − x₂| < δ → |f(x₁) − f(x₂)| < ε/(2(b−a))
  3. Разобьём [a,b] на n частей длины Δxₖ < δ. На каждой части колебание f менее ε/(2(b−a))
  4. Покрываем кривую прямоугольниками шириной Δxₖ и высотой < ε/(b−a)
  5. Суммарная площадь покрытия < Σ Δxₖ · ε/(b−a) = ε/(b−a) · (b−a) = ε
Следствие: область, ограниченная конечным числом непрерывных кривых, квадрируема (её граница имеет меру ноль).

Теорема 1.2. Свойства площади

  • Монотонность: F₁ ⊂ F₂ → S(F₁) ≤ S(F₂)
  • Аддитивность: S(F₁ ∪ F₂) = S(F₁) + S(F₂) (если F₁ и F₂ не пересекаются по внутренним точкам)
  • Инвариантность: при движениях (повороты, сдвиги) площадь сохраняется
  • Если F₁ и F₂ квадрируемы → F₁ ∩ F₂, F₁ ∪ F₂, F₁ \ F₂ тоже квадрируемы

Замечание 1.6. Мера Жордана

Площадь квадрируемой фигуры называется также мерой Жордана (в честь Камиля Жордана). Это конечно-аддитивная мера, определённая на кольце квадрируемых множеств. В отличие от меры Лебега, мера Жордана не счётно-аддитивна (например, множество рациональных точек отрезка имеет меру Лебега ноль, но не является квадрируемым по Жордану).

Двойной интеграл: определение, геометрический и физический смысл, свойства

Определение 2.1. Интегральная сумма Римана

Пусть G — замкнутая квадрируемая область на плоскости, f(x,y) определена и ограничена на G.

Разобьём G на n частей G₁, G₂, ..., Gₙ с площадями ΔS₁, ΔS₂, ..., ΔSₙ. В каждой части выберем произвольную точку (ξₖ, ηₖ) ∈ Gₖ.

Интегральная сумма Римана:

σ = Σₖ₌₁ⁿ f(ξₖ, ηₖ) · ΔSₖ

Определение 2.2. Ранг разбиения и двойной интеграл

Ранг разбиения (мелкость):

λ = max { d(Gₖ) : k = 1, ..., n }

где d(Gₖ) — диаметр k-й части (см. Определение 1.1).

Двойной интеграл — предел интегральной суммы при λ → 0:

∬_G f(x,y) dσ = lim (λ→0) Σₖ₌₁ⁿ f(ξₖ, ηₖ) · ΔSₖ

Предел должен существовать и не зависеть ни от способа разбиения, ни от выбора точек (ξₖ, ηₖ).

Замечание 2.1

  • 1) Интегрируемы только ограниченные функции на квадрируемых областях
  • 2) При f = 1: ∬_G 1 dσ = S(G) — площадь области G
  • 3) При f(x,y) ≥ 0: ∬_G f dσ = объём цилиндрического тела, ограниченного снизу областью G, сверху поверхностью z = f(x,y), с боков — цилиндрической поверхностью над ∂G

Геометрический смысл

Если f(x,y) ≥ 0 на G, то ∬_G f(x,y) dσ равен объёму цилиндрического тела с основанием G и «крышкой» z = f(x,y).

Если f меняет знак — получается алгебраический объём (с учётом знака: выше xOy — плюс, ниже — минус).

Физический смысл

Если f(x,y) = ρ(x,y) — поверхностная плотность пластинки G, то:

∬_G ρ(x,y) dσ = m — масса пластинки

Теорема 2.2. Свойства двойного интеграла

  • 1) Линейность: ∬(αf + βg)dσ = α∬f dσ + β∬g dσ (α, β — const)
  • 2) Аддитивность по области: если G = G₁ ∪ G₂, причём G₁ и G₂ не имеют общих внутренних точек:
    ∬_G f dσ = ∬_{G₁} f dσ + ∬_{G₂} f dσ
  • 3) Монотонность: если f(x,y) ≤ g(x,y) на G → ∬_G f dσ ≤ ∬_G g dσ
  • 4) Оценка интеграла:
    m · S(G) ≤ ∬_G f dσ ≤ M · S(G)

    где m = inf f, M = sup f на G, S(G) — площадь

  • 5) Теорема о среднем: если f непрерывна на замкнутой квадрируемой связной G, то ∃(ξ, η) ∈ G:
    ∬_G f dσ = f(ξ, η) · S(G)
  • 6) Оценка модуля: |∬_G f dσ| ≤ ∬_G |f| dσ

Теорема существования двойного интеграла. Условия интегрируемости

Теорема 2.1. Условия существования двойного интеграла

Условие 1. Если f(x,y) непрерывна на замкнутой квадрируемой области G, то f интегрируема на G (двойной интеграл существует).

Условие 2. Если f(x,y) непрерывна на G всюду, кроме множества меры ноль, и ограничена на G, то f интегрируема на G.

Условие 3. Изменение значений функции на множестве меры ноль не влияет на интегрируемость и значение интеграла.

Вспомни Определение 1.3: множество меры ноль — то, которое можно покрыть многоугольной фигурой сколь угодно малой площади. Например: конечное число точек, конечное число непрерывных кривых (по Лемме 1.1).

Связь с теорией площади (§1)

Существование двойного интеграла опирается на понятие квадрируемости области (Определение 1.2) и свойства множеств меры ноль. Вот почему §1 важен для этого билета:

  • Область G должна быть квадрируемой (т.е. её граница — множество меры ноль, по Следствию к Теореме 1.1)
  • Точки разрыва f должны образовывать множество меры ноль (Условие 2)
  • По Лемме 1.1 графики непрерывных функций имеют меру ноль → области, ограниченные непрерывными кривыми, квадрируемы

Необходимое условие

Если f интегрируема на G → f ограничена на G.

Первая теорема о среднем

Если f непрерывна на замкнутой связной квадрируемой G, то ∃ (x₀, y₀) ∈ G:

∬_G f(x,y) dσ = f(x₀, y₀) · S(G)

Доказательство: из оценки (свойство 4) m·S ≤ ∬f dσ ≤ M·S, делим на S(G) > 0:

m ≤ (1/S(G)) · ∬_G f dσ ≤ M

По теореме Больцано-Коши о промежуточном значении (f непрерывна на связном G) ∃ точка (x₀,y₀) ∈ G, в которой f принимает это среднее значение.

Вторая (обобщённая) теорема о среднем

Если f непрерывна на G и g(x,y) ≥ 0 интегрируема на G, то ∃ (x₀,y₀) ∈ G:

∬_G f · g dσ = f(x₀, y₀) · ∬_G g dσ
На экзамене по этому билету могут спросить: «Что такое квадрируемая область? Множество меры ноль?» — нужно знать Определения 1.2, 1.3, Лемму 1.1 из §1.

Сведение к повторному интегралу. Замена переменных

Область 1-го типа (правильная по y)

G задана неравенствами: a ≤ x ≤ b, φ₁(x) ≤ y ≤ φ₂(x)

где φ₁, φ₂ — непрерывные функции.

∬_G f(x,y) dσ = ∫ₐᵇ dx ∫_{φ₁(x)}^{φ₂(x)} f(x,y) dy

Порядок вычисления: сначала внутренний интеграл по y (при фиксированном x), затем внешний по x.

Область 2-го типа (правильная по x)

G задана: c ≤ y ≤ d, ψ₁(y) ≤ x ≤ ψ₂(y)

∬_G f(x,y) dσ = ∫_c^d dy ∫_{ψ₁(y)}^{ψ₂(y)} f(x,y) dx

Расстановка пределов (алгоритм)

  1. Нарисовать область G и определить её границы
  2. Определить тип (I или II) — провести «стрелку»: для типа I стрелка по y (вертикально), для типа II — по x (горизонтально)
  3. Внешний интеграл — по «свободной» переменной (постоянные пределы: числа a,b или c,d)
  4. Внутренний — по «зависимой» переменной (пределы — функции от внешней переменной)
  5. Если область не является ни типом I, ни типом II — разбить на подобласти

Замена переменных (общая формула)

Пусть x = x(u,v), y = y(u,v) — гладкое взаимно однозначное отображение. Тогда:

∬_G f(x,y) dxdy = ∬_{G'} f(x(u,v), y(u,v)) · |J| dudv

где J — якобиан перехода:

J = ∂(x,y)/∂(u,v) = | ∂x/∂u ∂x/∂v | = ∂x/∂u · ∂y/∂v − ∂x/∂v · ∂y/∂u | ∂y/∂u ∂y/∂v |

Условия применимости: отображение диффеоморфизм (гладкое, взаимно однозначное, J ≠ 0 всюду кроме, быть может, множества меры ноль).

Полярные координаты

x = r·cos φ, y = r·sin φ (r ≥ 0, 0 ≤ φ < 2π)

J = | cos φ −r·sin φ | = r·cos²φ + r·sin²φ = r | sin φ r·cos φ |
∬_G f(x,y) dxdy = ∬_{G'} f(r·cos φ, r·sin φ) · r · dr dφ
Полярные координаты удобны, когда: 1) Область — круг, полукруг, сектор, кольцо 2) Подынтегральное выражение содержит x² + y² 3) Граница задана уравнением вида r = f(φ)
Пример: обобщённые полярные (эллипс)

Область: x²/a² + y²/b² ≤ 1. Замена: x = a·r·cos φ, y = b·r·sin φ, |J| = a·b·r

Область в новых координатах: 0 ≤ r ≤ 1, 0 ≤ φ < 2π

S = ∬ dxdy = ∫₀^{2π} dφ ∫₀¹ a·b·r dr = 2π · a·b/2 = π·a·b

Вычисление двойных интегралов

Общий алгоритм

  1. Нарисовать область G, найти точки пересечения границ
  2. Определить тип области (I, II или разбить на части)
  3. Расставить пределы интегрирования
  4. Вычислить внутренний интеграл (внешняя переменная — параметр)
  5. Вычислить внешний интеграл
Пример 1: прямоугольная область

∬_G xy dxdy, G: 0 ≤ x ≤ 1, 0 ≤ y ≤ 2

Тип I: = ∫₀¹ dx ∫₀² xy dy = ∫₀¹ x · [y²/2]₀² dx = ∫₀¹ 2x dx = [x²]₀¹ = 1

Пример 2: треугольная область

∬_G x dxdy, G: x ≥ 0, y ≥ 0, x + y ≤ 1

Тип I: 0 ≤ x ≤ 1, 0 ≤ y ≤ 1−x

= ∫₀¹ dx ∫₀^{1−x} x dy = ∫₀¹ x·(1−x) dx = ∫₀¹ (x − x²) dx = [x²/2 − x³/3]₀¹ = 1/2 − 1/3 = 1/6

Пример 3: полярные координаты

∬_G (x² + y²) dxdy, G: x² + y² ≤ R²

Переход: x² + y² = r², dxdy = r·dr·dφ, область: 0 ≤ r ≤ R, 0 ≤ φ ≤ 2π

= ∫₀^{2π} dφ ∫₀^R r² · r dr = 2π · [r⁴/4]₀^R = πR⁴/2

Пример 4: смена порядка интегрирования

Дано: ∫₀¹ dx ∫ₓ¹ sin(y²) dy (внутренний не берётся в элементарных функциях!)

Область: 0 ≤ x ≤ 1, x ≤ y ≤ 1. Нарисуем: треугольник с вершинами (0,0), (1,1), (0,1).

Меняем порядок (тип II): 0 ≤ y ≤ 1, 0 ≤ x ≤ y

= ∫₀¹ dy ∫₀^y sin(y²) dx = ∫₀¹ y·sin(y²) dy = [−cos(y²)/2]₀¹ = (1 − cos 1)/2

Пример 5: область, ограниченная параболой и прямой

∬_G y dxdy, G: y = x², y = x (область между параболой и прямой)

Точки пересечения: x² = x → x = 0, x = 1

Тип I: 0 ≤ x ≤ 1, x² ≤ y ≤ x

= ∫₀¹ dx ∫_{x²}^{x} y dy = ∫₀¹ [y²/2]_{x²}^{x} dx = ∫₀¹ (x²/2 − x⁴/2) dx

= 1/2 · [x³/3 − x⁵/5]₀¹ = 1/2 · (1/3 − 1/5) = 1/2 · 2/15 = 1/15

Если внутренний интеграл не берётся в элементарных функциях — попробуй сменить порядок интегрирования (пример 4). Это классический приём на экзамене!

Приложения двойных интегралов (геометрические и физические)

Площадь плоской области

S = ∬_G 1 dxdy

Объём тела

Тело ограничено снизу z = f₁(x,y), сверху z = f₂(x,y), f₂ ≥ f₁, проекция на xOy — область G:

V = ∬_G (f₂(x,y) − f₁(x,y)) dxdy

Частный случай: тело под поверхностью z = f(x,y) ≥ 0 над областью G: V = ∬_G f dxdy

Площадь поверхности

Для поверхности z = f(x,y) над областью G:

S_пов = ∬_G √(1 + (∂f/∂x)² + (∂f/∂y)²) dxdy
Пример: площадь части полусферы

z = √(R² − x² − y²), G: x² + y² ≤ R²

∂z/∂x = −x/√(R²−x²−y²), ∂z/∂y = −y/√(R²−x²−y²)

1 + (∂z/∂x)² + (∂z/∂y)² = 1 + (x²+y²)/(R²−x²−y²) = R²/(R²−x²−y²)

S = ∬_G R/√(R²−x²−y²) dxdy

В полярных: = ∫₀^{2π} dφ ∫₀^R R·r/√(R²−r²) dr = 2π · R · [−√(R²−r²)]₀^R = 2πR²

Масса пластинки

Если ρ(x,y) — поверхностная плотность:

m = ∬_G ρ(x,y) dxdy

Статические моменты

Относительно координатных осей:

Mₓ = ∬_G y · ρ(x,y) dxdy (момент относительно оси Ox) My = ∬_G x · ρ(x,y) dxdy (момент относительно оси Oy)

Координаты центра тяжести

xc = My / m = ∬ x·ρ dσ / ∬ ρ dσ yc = Mₓ / m = ∬ y·ρ dσ / ∬ ρ dσ

Для однородной пластинки (ρ = const): xc = ∬x dσ / S, yc = ∬y dσ / S

Моменты инерции

Относительно координатных осей:

Iₓ = ∬_G y² · ρ dxdy (момент инерции относительно Ox) Iy = ∬_G x² · ρ dxdy (момент инерции относительно Oy)

Относительно начала координат (полярный момент):

I₀ = Iₓ + Iy = ∬_G (x² + y²) · ρ dxdy
Пример: центр тяжести треугольника

Однородный треугольник: x ≥ 0, y ≥ 0, x + y ≤ 1, ρ = 1

S = 1/2 (площадь)

xc = (1/S)·∬x dσ = 2·∫₀¹ dx ∫₀^{1−x} x dy = 2·∫₀¹ x(1−x) dx = 2·(1/2 − 1/3) = 1/3

По симметрии (относительно y=x): yc = 1/3

Центр тяжести: (1/3, 1/3) — совпадает с центроидом (пересечение медиан).

§3. Тройные интегралы

Тройной интеграл: определение, геометрический и физический смысл, свойства

Определение (аналогично двойному)

Пусть V — замкнутая ограниченная кубируемая область в R³, f(x,y,z) определена и ограничена на V.

  1. Разбиваем V на n частей V₁, V₂, ..., Vₙ с объёмами ΔV₁, ΔV₂, ..., ΔVₙ
  2. В каждой части выбираем точку (ξₖ, ηₖ, ζₖ) ∈ Vₖ
  3. Составляем интегральную сумму: σ = Σ f(ξₖ, ηₖ, ζₖ) · ΔVₖ
  4. Ранг разбиения: λ = max { d(Vₖ) }
∭_V f(x,y,z) dv = lim (λ→0) Σₖ₌₁ⁿ f(ξₖ, ηₖ, ζₖ) · ΔVₖ

Предел должен существовать и не зависеть от способа разбиения и выбора точек.

Геометрический смысл

∭_V 1 dv = объём области V

(аналог: ∬_G 1 dσ = площадь для двойного интеграла)

Физический смысл

Если f = ρ(x,y,z) — объёмная плотность тела V:

∭_V ρ(x,y,z) dv = m — масса тела

Свойства (аналогичны двойному интегралу)

  • Линейность: ∭(αf + βg) dv = α∭f dv + β∭g dv
  • Аддитивность по области: V = V₁ ∪ V₂ (без общих внутренних точек) → ∭_V = ∭_{V₁} + ∭_{V₂}
  • Монотонность: f ≤ g на V → ∭f dv ≤ ∭g dv
  • Оценка: m·|V| ≤ ∭f dv ≤ M·|V|, где m = inf f, M = sup f, |V| — объём
  • Теорема о среднем: f непрерывна на связной V → ∃(ξ,η,ζ) ∈ V: ∭f dv = f(ξ,η,ζ)·|V|
  • Оценка модуля: |∭f dv| ≤ ∭|f| dv

Теорема существования тройного интеграла. Основные теоремы

Теорема существования

Если f(x,y,z) непрерывна на замкнутой ограниченной кубируемой области V, то тройной интеграл существует.

Более общее условие: если f непрерывна всюду на V, кроме множества меры ноль в R³, и ограничена, то f интегрируема.

Понятие кубируемости

Область V ⊂ R³ называется кубируемой (измеримой по Жордану в R³), если внутренний и внешний объёмы совпадают (аналогично квадрируемости в R²).

V кубируема ⟺ граница ∂V имеет меру ноль (трёхмерный аналог Следствия к Теореме 1.1).

Теорема о среднем

Если f непрерывна на замкнутой связной кубируемой V, то ∃ точка (x₀, y₀, z₀) ∈ V:

∭_V f(x,y,z) dv = f(x₀, y₀, z₀) · |V|

где |V| — объём области V.

Перенос свойств с двойного интеграла

Все основные теоремы двойного интеграла переносятся на тройной с естественными заменами:

  • Область G (плоская) → V (пространственная)
  • Площадь S(G) → Объём |V|
  • dσ (элемент площади) → dv (элемент объёма)
  • Квадрируемость → Кубируемость
  • Множество меры ноль в R² → Множество меры ноль в R³
Изменение значений f на множестве меры ноль в R³ не влияет на значение тройного интеграла (аналог Условия 3 Теоремы 2.1).

Сведение к повторному интегралу. Замена переменных

Сведение тройного интеграла к повторному

ВАЖНО (указание Петровой): Тройной интеграл ОБЯЗАТЕЛЬНО сначала выражается через ДВОЙНОЙ по проекции! НЕЛЬЗЯ сразу писать три определённых интеграла. Правильная запись — сначала сведение к двойному, потом двойной раскрывается в повторный.

Правильная запись (через двойной по проекции):

Пусть V проектируется на плоскость xOy в область G, и для каждой точки (x,y) ∈ G переменная z меняется от z₁(x,y) до z₂(x,y):

∭_V f dv = ∬_G [ ∫_{z₁(x,y)}^{z₂(x,y)} f(x,y,z) dz ] dxdy

Затем двойной интеграл по G раскрывается в повторный (по правилам §2).

Полная запись (если G — область типа I: a ≤ x ≤ b, φ₁(x) ≤ y ≤ φ₂(x)):

∭_V f dv = ∫ₐᵇ dx ∫_{φ₁(x)}^{φ₂(x)} dy ∫_{z₁(x,y)}^{z₂(x,y)} f(x,y,z) dz

Расстановка пределов (алгоритм)

  1. Определить проекцию G тела V на плоскость xOy (или xOz, yOz — в зависимости от формы тела)
  2. Для каждой точки (x,y) ∈ G определить пределы по z: от z₁(x,y) до z₂(x,y)
  3. В области G расставить пределы по x и y (как для двойного интеграла)

Замена переменных (общая формула)

Пусть x = x(u,v,w), y = y(u,v,w), z = z(u,v,w) — гладкое взаимно однозначное отображение:

∭_V f(x,y,z) dxdydz = ∭_{V'} f(...) · |J| dudvdw

J = ∂(x,y,z)/∂(u,v,w) — якобиан (определитель матрицы 3×3 частных производных).

Цилиндрические координаты

x = r·cos φ,   y = r·sin φ,   z = z

r ≥ 0, 0 ≤ φ < 2π, z ∈ R

|J| = r   →   dxdydz = r · dr dφ dz

По сути: полярные координаты в плоскости xOy + обычная координата z.

Цилиндрические координаты удобны для тел с осью симметрии (ось Oz): цилиндры, конусы, параболоиды вращения.

Сферические координаты

x = r·sin θ·cos φ,   y = r·sin θ·sin φ,   z = r·cos θ

r ≥ 0 (расстояние от начала координат), 0 ≤ θ ≤ π (полярный угол от оси Oz), 0 ≤ φ < 2π (азимутальный угол)

|J| = r²·sin θ   →   dxdydz = r²·sin θ · dr dθ dφ
Сферические координаты удобны для шаров и конусов с вершиной в начале координат. Ключевое соотношение: x² + y² + z² = r².
Запоминалка: якобианы

Полярные (2D): |J| = r

Цилиндрические (3D): |J| = r

Сферические (3D): |J| = r²·sin θ

Вычисление тройных интегралов

Общий алгоритм

  1. Описать и (хотя бы схематично) нарисовать область V
  2. Выбрать систему координат (декартовы, цилиндрические, сферические)
  3. Найти проекцию G на подходящую координатную плоскость
  4. Записать тройной интеграл через двойной по проекции (указание Петровой!)
  5. Раскрыть двойной в повторный и расставить пределы
  6. Вычислить последовательно: сначала внутренний, потом средний, потом внешний
Пример 1: параллелепипед (декартовы)

∭_V xyz dv, V: 0 ≤ x ≤ 1, 0 ≤ y ≤ 2, 0 ≤ z ≤ 3

Проекция на xOy: G = {0 ≤ x ≤ 1, 0 ≤ y ≤ 2}

∭ xyz dv = ∬_G [∫₀³ xyz dz] dxdy = ∬_G xy·[z²/2]₀³ dxdy = ∬_G (9/2)xy dxdy

= (9/2)·∫₀¹ dx ∫₀² xy dy = (9/2)·∫₀¹ x·[y²/2]₀² dx = (9/2)·∫₀¹ 2x dx = (9/2)·1 = 9/2

Пример 2: объём шара (сферические координаты)

∭_V dv, V: x² + y² + z² ≤ R²

Сферические: r от 0 до R, θ от 0 до π, φ от 0 до 2π

= ∫₀^{2π} dφ ∫₀^π sin θ dθ ∫₀^R r² dr

= 2π · [−cos θ]₀^π · [r³/3]₀^R = 2π · 2 · R³/3 = 4πR³/3

Пример 3: цилиндрические координаты

∭_V z dv, V: x² + y² ≤ 1, 0 ≤ z ≤ 1 (цилиндр)

Проекция на xOy: круг r ≤ 1. В цилиндрических:

∭ z dv = ∬_G [∫₀¹ z dz] · r dr dφ = ∬_G (1/2) · r dr dφ

= (1/2)·∫₀^{2π} dφ ∫₀¹ r dr = (1/2)·2π·(1/2) = π/2

Пример 4: тело под параболоидом (цилиндрические)

∭_V dv, V: z ≥ 0, z ≤ 1 − x² − y²

Проекция на xOy: x² + y² ≤ 1 (круг). В цилиндрических: z от 0 до 1−r²

= ∬_G [∫₀^{1−r²} dz] · r dr dφ = ∬_G (1−r²)·r dr dφ

= ∫₀^{2π} dφ ∫₀¹ (r − r³) dr = 2π · [r²/2 − r⁴/4]₀¹ = 2π · 1/4 = π/2

Пример 5: конус (сферические)

∭_V (x²+y²+z²) dv, V: конус θ ≤ α, 0 ≤ r ≤ R

В сферических: x²+y²+z² = r²

= ∫₀^{2π} dφ ∫₀^α sin θ dθ ∫₀^R r² · r² dr

= 2π · (1 − cos α) · R⁵/5

Частая ошибка: сразу писать ∫dx∫dy∫dz (три определённых интеграла) вместо того, чтобы сначала выделить двойной по проекции. На экзамене за это снижают!

Приложения тройных интегралов (геометрические и физические)

Объём тела

V = ∭_V 1 dxdydz

Масса тела

ρ(x,y,z) — объёмная плотность:

m = ∭_V ρ(x,y,z) dv

Статические моменты

Относительно координатных плоскостей:

Mxy = ∭_V z · ρ dv (момент относительно пл. xOy) Mxz = ∭_V y · ρ dv (момент относительно пл. xOz) Myz = ∭_V x · ρ dv (момент относительно пл. yOz)

Координаты центра масс

xc = Myz / m = ∭ x·ρ dv / m yc = Mxz / m = ∭ y·ρ dv / m zc = Mxy / m = ∭ z·ρ dv / m

Для однородного тела (ρ = const): xc = ∭x dv / V, аналогично yc, zc.

Моменты инерции

Относительно координатных осей:

Ix = ∭_V (y² + z²) · ρ dv Iy = ∭_V (x² + z²) · ρ dv Iz = ∭_V (x² + y²) · ρ dv

Логика: момент инерции относительно оси = ∭ (расстояние до оси)² · ρ dv. Расстояние от точки (x,y,z) до оси Oz = √(x²+y²), поэтому Iz = ∭(x²+y²)ρ dv.

Относительно координатных плоскостей:

Ixy = ∭_V z² · ρ dv Ixz = ∭_V y² · ρ dv Iyz = ∭_V x² · ρ dv

Относительно начала координат:

I₀ = ∭_V (x² + y² + z²) · ρ dv = Iyz + Ixz + Ixy
Связь между моментами: Ix = Ixz + Ixy, Iy = Iyz + Ixy, Iz = Iyz + Ixz. Также: I₀ = (Ix + Iy + Iz) / 2.
Пример 1: центр масс однородного полушара

V: x² + y² + z² ≤ R², z ≥ 0, ρ = const

По симметрии: xc = 0, yc = 0 (тело симметрично относительно оси Oz).

V = (2/3)πR³ (объём полушара)

В сферических координатах:

∭ z dv = ∫₀^{2π} dφ ∫₀^{π/2} cos θ · sin θ dθ ∫₀^R r³ dr

= 2π · [sin²θ / 2]₀^{π/2} · [r⁴/4]₀^R = 2π · 1/2 · R⁴/4 = πR⁴/4

zc = (πR⁴/4) / ((2/3)πR³) = 3R/8

Пример 2: момент инерции однородного шара относительно оси Oz

V: x² + y² + z² ≤ R², ρ = const

Iz = ρ · ∭(x²+y²) dv. В сферических: x²+y² = r²·sin²θ

Iz = ρ · ∫₀^{2π} dφ ∫₀^π sin²θ · sinθ dθ ∫₀^R r⁴ dr

= ρ · 2π · ∫₀^π sin³θ dθ · R⁵/5

∫₀^π sin³θ dθ = 4/3, поэтому:

Iz = ρ · 2π · (4/3) · R⁵/5 = 8πρR⁵/15

Подставляя m = ρ · (4/3)πR³:

Iz = (2/5)mR²

Пример 3: момент инерции однородного цилиндра

V: x² + y² ≤ R², 0 ≤ z ≤ H, ρ = const

Iz = ρ · ∭(x²+y²) dv. В цилиндрических: x²+y² = r²

= ρ · ∫₀^{2π} dφ ∫₀^R r² · r dr ∫₀^H dz = ρ · 2π · R⁴/4 · H = πρR⁴H/2

Подставляя m = ρπR²H: Iz = (1/2)mR²

При наличии симметрии сразу определяй, какие координаты центра масс равны нулю — это значительно экономит время на экзамене! Например: тело симметрично относительно оси Oz → xc = yc = 0.
