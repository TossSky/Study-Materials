Задачи, приводящие к дифференциальным уравнениям

Что такое дифференциальное уравнение

Определение 1.1. Обыкновенное дифференциальное уравнение (ОДУ) n-го порядка — это уравнение вида:

F(x, y, y', y'', ..., y⁽ⁿ⁾) = 0

где x — независимая переменная, y(x) — неизвестная функция, y', ..., y⁽ⁿ⁾ — её производные. Слово «обыкновенное» означает, что функция зависит от одной переменной (в отличие от уравнений в частных производных).

Определение 1.2. Решение ДУ — функция y = φ(x), определённая на интервале (a, b), имеющая производные до порядка n включительно и удовлетворяющая уравнению тождественно при всех x из (a, b).

Определение 1.3. График решения y = φ(x) на плоскости (x, y) называется интегральной кривой.

Откуда берутся ДУ

Многие законы природы описывают скорость изменения величины — а скорость изменения это производная. Формулируя закон математически, получаем ДУ.

Классические примеры

1. Радиоактивный распад: скорость распада пропорциональна количеству вещества:

dN/dt = −λN → N(t) = N₀ · e^(−λt)

ДУ 1-го порядка. Решение — экспоненциальный закон убывания.

2. Закон охлаждения Ньютона: скорость изменения температуры тела пропорциональна разности температур тела и среды:

dT/dt = −k(T − T_окр)

ДУ 1-го порядка с разделяющимися переменными.

3. Второй закон Ньютона: F = ma = m·x'', где x(t) — координата. Для пружины (F = −kx):

mx'' + kx = 0 → x'' + ω²x = 0, ω² = k/m

ДУ 2-го порядка. Решение: гармонические колебания x = A cos(ωt + φ).

4. Рост популяции (модель Мальтуса):

dP/dt = rP → P(t) = P₀ · e^(rt)

5. Геометрическая задача: найти кривую, у которой длина подкасательной постоянна. Подкасательная = y/y', отсюда:

y / y' = a → y' = y/a
Вывод: задачи физики, химии, биологии, геометрии формулируются через производные и приводят к дифференциальным уравнениям. Решить ДУ — значит найти функцию по информации о её производных.
2

ОДУ 1-го порядка: общее/частное решение, задача Коши, теорема существования и единственности

Общий вид ДУ 1-го порядка

F(x, y, y') = 0

Если можно разрешить относительно y', получаем нормальную форму (Определение 2.2):

y' = f(x, y)

Порядок = 1, потому что старшая производная — y'.

Общий интеграл (Определение 2.1)

Соотношение Ф(x, y, C) = 0 называется общим интегралом ДУ, если при дифференцировании по x и исключении произвольной постоянной C получается исходное ДУ.

Теорема 2.1. Если Ф разрешимо относительно C, т.е. C = Ψ(x, y), то все решения ДУ содержатся в общем интеграле.

Общее решение

y = φ(x, C)

Семейство кривых с одной произвольной постоянной C. Каждому значению C соответствует одна интегральная кривая.

Частное решение

Получается из общего при конкретном C. Пример: y = Ce^x — общее, y = 3e^x — частное (C = 3).

Особое решение

Решение, которое НЕ получается из общего ни при каком C. Может возникать при делении на выражение, равное нулю. Пример: для y' = 2√y общее решение y = (x+C)², но y = 0 — особое решение.

Задача Коши (Определение 1.5)

Найти решение ДУ, удовлетворяющее начальному условию:

y' = f(x, y), y(x₀) = y₀

Геометрически: из всего семейства интегральных кривых выбрать ту, которая проходит через точку (x₀, y₀).

Теорема 2.2 (существования и единственности)

Пусть функция f(x, y) и её частная производная f'_y непрерывны в некоторой области G на плоскости (x, y). Тогда для любой точки (x₀, y₀) ∈ G задача Коши имеет решение, и притом единственное.

Иными словами: через каждую точку области G проходит ровно одна интегральная кривая.

Если условия теоремы 2.2 нарушены (например, f'_y не существует или не непрерывна), решение может быть не единственным. Классический пример: y' = y^(1/3) — через точку (0, 0) проходят бесконечно много решений, т.к. f'_y = (1/3)y^(-2/3) → ∞ при y → 0.
На экзамене часто просят: «Сформулируйте теорему существования и единственности». Нужно чётко назвать условия (непрерывность f и f'_y) и заключение (единственное решение задачи Коши).
3

Геометрическая интерпретация ДУ 1-го порядка. Метод изоклин

Поле направлений

Уравнение y' = f(x, y) задаёт в каждой точке (x, y) плоскости значение производной — то есть наклон касательной к интегральной кривой, проходящей через эту точку.

Рисуем в каждой точке маленький отрезок с наклоном f(x, y) — получаем поле направлений. Интегральные кривые — линии, которые в каждой точке касаются этих отрезков.

Если условия теоремы 2.2 выполнены, интегральные кривые не пересекаются (единственность решения задачи Коши).

Определение изоклины

Изоклина — кривая на плоскости, во всех точках которой наклон поля направлений одинаков.

f(x, y) = k (k = const)

Во всех точках изоклины касательные к интегральным кривым имеют один и тот же угловой коэффициент k. Семейство изоклин (при различных k) позволяет восстановить качественную картину поля направлений.

Метод изоклин — построение интегральных кривых

  1. Задаём набор значений k: 0, ±1, ±2, 1/2, ...
  2. Для каждого k строим кривую f(x, y) = k — это изоклина
  3. На каждой изоклине рисуем короткие отрезки с наклоном k
  4. Проводим интегральные кривые, «следуя» полю направлений от изоклины к изоклине
Пример: y' = x + y

Изоклины: x + y = k — семейство прямых y = −x + k

k = 0: прямая y = −x → горизонтальные отрезки (наклон 0)

k = 1: прямая y = −x + 1 → отрезки с наклоном 1 (под 45°)

k = −1: прямая y = −x − 1 → отрезки с наклоном −1

Соединяя, видим, что интегральные кривые «закручиваются» вверх и вправо.

Пример: y' = −x/y (окружности)

Изоклины: −x/y = k → y = −x/k — прямые через начало координат.

Интегральные кривые: окружности x² + y² = C² (центр в начале).

Особые точки

Точки, где f(x, y) не определена или разрывна — особые точки поля. Через них теорема единственности может не работать, и интегральные кривые могут «сливаться» или «расходиться».

Метод изоклин даёт качественную картину решений без аналитического решения ДУ. На экзамене могут попросить нарисовать поле направлений и несколько интегральных кривых для конкретного уравнения.

ДУ с разделяющимися переменными

Определение (параграф 2, тип I)

ДУ с разделяющимися переменными — уравнение, правая часть которого есть произведение функции от x на функцию от y:

y' = g₁(x) · g₂(y)

В дифференциальной форме: M(x)·N(y)dx + P(x)·Q(y)dy = 0

Ключевой признак: переменные x и y можно «развести» по разным сторонам равенства.

Метод решения

  1. Если g₂(y) ≠ 0, делим обе части на g₂(y): dy/g₂(y) = g₁(x)dx
  2. Интегрируем обе части: ∫ dy/g₂(y) = ∫ g₁(x)dx + C
  3. Если возможно, выражаем y явно через x

Общее решение (общий интеграл)

∫ dy/g₂(y) = ∫ g₁(x)dx + C

Дополнительные (особые) решения

Если g₂(c*) = 0 для некоторого c*, то y = c* — постоянное решение (стационарная точка). Оно может быть особым (не получается из общего интеграла ни при каком C).

При разделении переменных мы делим на g₂(y). Нужно отдельно проверить корни g₂(y) = 0 — они дают дополнительные решения, которые легко потерять!
Пример

y' = 2x · y²

Разделяем: dy/y² = 2x dx (при y ≠ 0)

Интегрируем: −1/y = x² + C

Общее решение: y = −1/(x² + C)

Особое решение: y = 0 (потеряли при делении на y²)

Пример 2: задача Коши

y' = y·sin x, y(0) = 1

Разделяем: dy/y = sin x dx

Интегрируем: ln|y| = −cos x + C → y = A·e^(−cos x)

Из y(0) = 1: 1 = A·e^(−1) → A = e

Ответ: y = e^(1 − cos x)

Однородные ДУ 1-го порядка

Однородная функция

Функция f(x, y) называется однородной степени k, если:

f(tx, ty) = t^k · f(x, y) для любого t > 0

Например: f(x,y) = x² + xy — однородная степени 2, т.к. f(tx,ty) = t²x² + t²xy = t²·f(x,y).

Вид однородного ДУ

ДУ y' = f(x, y) называется однородным, если f(x, y) — однородная функция степени 0:

y' = φ(y/x)

Как распознать: если заменить y на ty, x на tx, правая часть не изменится. Эквивалентно: в дроби все слагаемые числителя и знаменателя имеют одинаковую суммарную степень по x и y.

Метод решения — замена t = y/x

  1. Замена: t = y/x, тогда y = t·x, y' = t + t'·x
  2. Подставляем в ДУ: t + t'·x = φ(t)
  3. Выражаем: t'·x = φ(t) − t
  4. Разделяем переменные: dt / (φ(t) − t) = dx / x
  5. Интегрируем обе части
  6. Обратная замена: t = y/x

Общий интеграл

∫ dt/(φ(t) − t) = ln|x| + C, затем t = y/x
После замены t = y/x всегда получаем ДУ с разделяющимися переменными. Не забудь проверить: если φ(t₀) − t₀ = 0, то t = t₀ (т.е. y = t₀·x) — дополнительное решение.
Пример

y' = (x² + y²) / (2xy)

Проверка однородности: степень числителя = 2, степень знаменателя = 2 → однородное ДУ.

Замена t = y/x: t + t'x = (1 + t²) / (2t)

t'x = (1 + t²)/(2t) − t = (1 + t² − 2t²)/(2t) = (1 − t²)/(2t)

Разделяем: 2t·dt / (1 − t²) = dx/x

Интегрируем: −ln|1 − t²| = ln|x| + C₁

Упрощаем: |x| / |1 − t²| = C → x / (1 − y²/x²) = C → x² − y² = Cx

Формула для быстрой проверки: в уравнении M(x,y)dx + N(x,y)dy = 0, если M и N однородны одной и той же степени — уравнение однородное.

Линейные ДУ 1-го порядка

Определение (параграф 2, тип II)

y' + p(x)·y = q(x)

Линейность: y и y' входят в первой степени и не перемножаются друг с другом. Коэффициенты p(x) и q(x) зависят только от x.

Если q(x) ≡ 0 — линейное однородное (ЛОДУ). Если q(x) ≢ 0 — линейное неоднородное (ЛНДУ).

Решение ЛОДУ: y' + p(x)·y = 0

Это ДУ с разделяющимися переменными:

dy/y = −p(x)dx → ln|y| = −∫p(x)dx + C₁
y_одн = C · e^(−∫p(x)dx) = C · e^(Φ(x)), где Φ(x) = −∫p(x)dx

Решение ЛНДУ: метод вариации постоянной (Лагранж)

Знаем решение однородного: y_одн = C·e^(Φ(x)). Заменяем C на функцию C(x):

  1. Пусть y = C(x) · e^(Φ(x))
  2. Подставляем в ЛНДУ: C'(x)·e^Φ + C(x)·Φ'·e^Φ + p(x)·C(x)·e^Φ = q(x)
  3. Поскольку Φ' = −p(x), два последних слагаемых сокращаются: C'(x)·e^Φ = q(x)
  4. C'(x) = q(x)·e^(−Φ(x)) = q(x)·e^(∫p(x)dx)
  5. Интегрируем: C(x) = ∫ q(x)·e^(∫p dx) dx + C*

Общее решение ЛНДУ

y(x) = e^(−∫p dx) · ( ∫ q·e^(∫p dx) dx + C* )

Или эквивалентно: y = (F(x) + C*) · e^(Φ(x)), где F(x) = ∫ q·e^(−Φ) dx.

Метод Бернулли (подстановка y = u·v)

Альтернативный способ — ищем решение как произведение двух функций:

  1. Пусть y = u·v, тогда y' = u'v + uv'
  2. Подставляем: u'v + u(v' + p(x)·v) = q(x)
  3. Выбираем v из условия v' + p(x)·v = 0 → v = e^(−∫p dx)
  4. Остаётся: u'·v = q(x) → u' = q(x)/v → интегрируем
  5. Ответ: y = u · v
Пример

y' − 2y = x (p = −2, q = x)

Однородное: y_одн = C·e^(2x)

Вариация: y = C(x)·e^(2x), подставляем: C'(x)·e^(2x) = x → C'(x) = x·e^(−2x)

Интегрируем по частям: C(x) = −x·e^(−2x)/2 − e^(−2x)/4 + C*

Ответ: y = (−x·e^(−2x)/2 − e^(−2x)/4 + C*)·e^(2x) = −x/2 − 1/4 + C*·e^(2x)

Не путай с разделением переменных! Слагаемое p(x)·y содержит y, поэтому переменные не разделяются (кроме однородного случая q = 0).

Уравнения Бернулли

Определение

y' + P(x)·y = Q(x)·yⁿ (n ≠ 0, n ≠ 1)

При n = 0 получаем линейное ДУ. При n = 1 получаем ДУ с разделяющимися переменными. Уравнение Бернулли — обобщение линейного ДУ, где правая часть содержит степень неизвестной функции.

Метод решения — замена z = y^(1−n)

  1. Делим обе части на yⁿ (при y ≠ 0): y⁻ⁿ·y' + P(x)·y^(1−n) = Q(x)
  2. Замена: z = y^(1−n), тогда z' = (1−n)·y⁻ⁿ·y', значит y⁻ⁿ·y' = z'/(1−n)
  3. Получаем линейное ДУ: z'/(1−n) + P(x)·z = Q(x)
  4. Т.е.: z' + (1−n)·P(x)·z = (1−n)·Q(x) — решаем по билету 6
  5. Обратная замена: y = z^(1/(1−n))
При делении на yⁿ мы предполагаем y ≠ 0. Нужно отдельно проверить, является ли y = 0 решением исходного уравнения (это так при n > 0).
Пример

y' + y = y² (P = 1, Q = 1, n = 2)

Делим на y²: y⁻²·y' + y⁻¹ = 1

Замена z = y⁻¹ = y^(1−2), z' = −y⁻²·y': −z' + z = 1 → z' − z = −1

Линейное ДУ: z_одн = Ce^x. Вариация → частное: z* = 1.

Общее: z = Ce^x + 1

Обратная замена: y = 1/z = 1/(Ce^x + 1)

Дополнительно: y = 0 — тоже решение (получается при C → ∞).

Пример 2

y' − y/x = x·y³ (P = −1/x, Q = x, n = 3)

Замена z = y^(−2), z' = −2y⁻³y'.

Линейное: z' + 2z/x = −2x

Решаем методом вариации постоянной.

Схема: Бернулли → делим на yⁿ → замена z = y^(1−n) → линейное ДУ → решаем → обратная замена.

ДУ в полных дифференциалах

Определение (параграф 2, тип III)

P(x, y)dx + Q(x, y)dy = 0

Уравнение называется уравнением в полных дифференциалах, если левая часть является полным дифференциалом некоторой функции F(x, y), т.е. dF = P·dx + Q·dy.

Критерий полного дифференциала (Лемма 2.1)

Лемма 2.1. В односвязной области D выражение P·dx + Q·dy является полным дифференциалом тогда и только тогда, когда:

∂P/∂y = ∂Q/∂x

при условии, что P, Q и их частные производные непрерывны в D.

Почему это работает

Если ∃ F(x, y) такая, что ∂F/∂x = P и ∂F/∂y = Q, то по теореме о равенстве смешанных производных:

∂²F/∂y∂x = ∂P/∂y = ∂Q/∂x = ∂²F/∂x∂y

Решение уравнения dF = 0 — это F(x, y) = C.

Метод нахождения F(x, y)

  1. Из условия ∂F/∂x = P интегрируем по x: F = ∫ P(x, y) dx + φ(y) (φ(y) — «постоянная» интегрирования, зависящая от y)
  2. Дифференцируем полученное F по y: ∂F/∂y = ∂/∂y(∫P dx) + φ'(y)
  3. Приравниваем к Q: ∂/∂y(∫P dx) + φ'(y) = Q → находим φ'(y)
  4. Интегрируем φ'(y) → φ(y)
  5. Ответ: F(x, y) = C
Можно начать и с Q: F = ∫ Q dy + ψ(x), затем определить ψ(x) из условия ∂F/∂x = P. Результат будет тот же.
Пример

(3x² + y)dx + (x + 2y)dy = 0

Проверка: ∂P/∂y = 1, ∂Q/∂x = 1 → равны, это полный дифференциал.

F = ∫(3x² + y)dx = x³ + xy + φ(y)

∂F/∂y = x + φ'(y) = x + 2y → φ'(y) = 2y → φ(y) = y²

Ответ: x³ + xy + y² = C

Интегрирующий множитель

Если условие ∂P/∂y ≠ ∂Q/∂x, можно попытаться найти функцию μ(x, y) такую, что уравнение μP·dx + μQ·dy = 0 будет в полных дифференциалах.

Частные случаи:

  • Если (∂P/∂y − ∂Q/∂x) / Q зависит только от x → μ = μ(x)
  • Если (∂Q/∂x − ∂P/∂y) / P зависит только от y → μ = μ(y)
На экзамене: сначала проверь условие ∂P/∂y = ∂Q/∂x. Если не выполнено — это НЕ уравнение в полных дифференциалах (может потребоваться интегрирующий множитель или другой метод).

ДУ высшего порядка: основные понятия, задача Коши, теорема существования

Общий вид (Определение 1.1 для n-го порядка)

F(x, y, y', y'', ..., y⁽ⁿ⁾) = 0

Порядок ДУ = порядок старшей производной.

В разрешённом (нормальном) виде:

y⁽ⁿ⁾ = f(x, y, y', ..., y⁽ⁿ⁻¹⁾)

Общее решение

y = φ(x, C₁, C₂, ..., Cₙ)

Содержит n произвольных постоянных — столько же, каков порядок ДУ.

Частное решение

Получается из общего при конкретных значениях C₁, ..., Cₙ.

Задача Коши (Определение 1.5)

Найти решение ДУ n-го порядка, удовлетворяющее начальным условиям:

y(x₀) = y₀, y'(x₀) = y₁, ..., y⁽ⁿ⁻¹⁾(x₀) = yₙ₋₁

Всего n условий — ровно столько, чтобы определить n произвольных постоянных.

Теорема 3.1 (существования и единственности)

Пусть функция f(x, y, y', ..., y⁽ⁿ⁻¹⁾) и все её частные производные ∂f/∂y, ∂f/∂y', ..., ∂f/∂y⁽ⁿ⁻¹⁾ непрерывны в окрестности точки (x₀, y₀, y₁, ..., yₙ₋₁). Тогда задача Коши имеет решение, и притом единственное.

Это прямое обобщение теоремы 2.2 (билет 2) на случай n-го порядка. Условия: непрерывность f и всех ∂f/∂y⁽ᵏ⁾.

Теорема 3.2 (для линейных ДУ)

Для линейного ДУ y⁽ⁿ⁾ + aₙ₋₁(x)y⁽ⁿ⁻¹⁾ + ... + a₀(x)y = q(x) с непрерывными коэффициентами aₖ(x) и правой частью q(x) на интервале (a, b), решение задачи Коши существует и единственно на всём интервале (a, b).

Ключевое отличие от нелинейного случая: для линейных ДУ решение существует на ВСЁМ промежутке непрерывности коэффициентов, а не только в окрестности начальной точки.

Понижение порядка ДУ (3 типа)

Тип 1: y⁽ⁿ⁾ = f(x)

В уравнении нет y, y', ..., y⁽ⁿ⁻¹⁾ — только x.

Метод: последовательно интегрируем n раз, на каждом шаге добавляя произвольную постоянную.

Пример

y'' = 6x

Первое интегрирование: y' = 3x² + C₁

Второе интегрирование: y = x³ + C₁x + C₂

Тип 2: F(x, y⁽ᵏ⁾, y⁽ᵏ⁺¹⁾, ..., y⁽ⁿ⁾) = 0 — нет младших производных

Уравнение не содержит y, y', ..., y⁽ᵏ⁻¹⁾ (нет функции и её младших производных).

Метод: замена p = y⁽ᵏ⁾. Тогда y⁽ᵏ⁺¹⁾ = p', y⁽ᵏ⁺²⁾ = p'', и т.д. Порядок снижается с n до n − k.

Пример

x·y'' = y' + x³ (нет y явно → тип 2, k = 1)

Замена p = y', p' = y'': x·p' = p + x³ → p' − p/x = x²

Это линейное ДУ 1-го порядка относительно p(x). Решаем методом вариации:

p_одн = Cx. Вариация: p = C(x)·x, C'(x)·x = x², C'(x) = x, C(x) = x²/2 + C₁.

p = y' = (x²/2 + C₁)·x = x³/2 + C₁x

Интегрируем: y = x⁴/8 + C₁x²/2 + C₂

Тип 3: F(y, y', y'', ..., y⁽ⁿ⁾) = 0 — нет x явно

Независимая переменная x не входит явно.

Метод: замена y' = p(y) — считаем p функцией от y (а не от x). Тогда:

y'' = dp/dx = (dp/dy)·(dy/dx) = p · dp/dy

Для третьей производной: y''' = (p·p'' + (p')²)·p, где штрих означает d/dy. Порядок снижается на 1.

Пример

y·y'' = (y')² (нет x → тип 3)

Замена y' = p, y'' = p·dp/dy: y·p·dp/dy = p²

При p ≠ 0 сокращаем на p: y·dp/dy = p

Разделяем: dp/p = dy/y → ln|p| = ln|y| + C₁ → p = C₁·y

Т.е. y' = C₁·y → dy/y = C₁dx → y = A·e^(C₁x)

Проверка p = 0 → y' = 0 → y = const (тоже решение).

Как определить тип: нет y и младших производных → тип 1 или 2; нет x → тип 3. Тип 1 — частный случай типа 2 при k = n.

ЛОДУ n-го порядка. Свойства решений (теорема 3.2)

Общий вид ЛОДУ

y⁽ⁿ⁾ + aₙ₋₁(x)·y⁽ⁿ⁻¹⁾ + ... + a₁(x)·y' + a₀(x)·y = 0

Все коэффициенты aₖ(x) непрерывны на интервале (a, b). Запишем кратко: L[y] = 0, где L — линейный дифференциальный оператор.

Теорема 3.2 (существование и единственность для ЛДУ)

Если коэффициенты aₖ(x) и правая часть q(x) непрерывны на (a, b), то для любой точки x₀ ∈ (a, b) и любых начальных данных y₀, y₁, ..., yₙ₋₁ задача Коши имеет единственное решение, определённое на всём интервале (a, b).

Принцип суперпозиции

Оператор L линеен, поэтому:

  • Если y₁ — решение L[y] = 0 и y₂ — решение L[y] = 0, то y = α·y₁ + β·y₂ — тоже решение L[y] = 0 при любых константах α, β.
  • В общем виде: линейная комбинация y = α₁y₁ + α₂y₂ + ... + αₖyₖ любого числа решений ЛОДУ — тоже решение.

Доказательство: L[α₁y₁ + α₂y₂] = α₁·L[y₁] + α₂·L[y₂] = α₁·0 + α₂·0 = 0 (по линейности оператора L).

Принцип суперпозиции не выполняется для неоднородных уравнений! Если y₁ и y₂ — решения L[y] = q(x), то y₁ + y₂ — НЕ решение (т.к. L[y₁ + y₂] = 2q(x) ≠ q(x)).

Множество решений ЛОДУ

Множество всех решений ЛОДУ образует линейное пространство (подпространство пространства всех функций). Его размерность равна n (порядку уравнения) — это будет доказано в теореме о существовании ФСР (билет 13).

Принцип суперпозиции — фундаментальное свойство линейных уравнений. Именно оно позволяет строить общее решение как линейную комбинацию базисных решений.

Линейная зависимость. Определитель Вронского (теорема 4.1)

Линейная зависимость функций (Определение 4.1)

Функции φ₁, φ₂, ..., φₙ называются линейно зависимыми (ЛЗ) на (a, b), если существуют числа α₁, ..., αₙ, не все равные нулю, такие что:

α₁φ₁(x) + α₂φ₂(x) + ... + αₙφₙ(x) ≡ 0 на (a, b)

Линейно независимы (ЛНЗ) — если такое тождество возможно только при α₁ = α₂ = ... = αₙ = 0.

Эквивалентно: функции ЛЗ ⟺ одна из них выражается через остальные.

Определитель Вронского (Определение 4.3)

Для n функций φ₁, ..., φₙ, имеющих производные до (n−1)-го порядка:

W(x) = det | φ₁ φ₂ ... φₙ | | φ₁' φ₂' ... φₙ' | | ... | | φ₁⁽ⁿ⁻¹⁾ φ₂⁽ⁿ⁻¹⁾... φₙ⁽ⁿ⁻¹⁾ |

Матрица размера n×n, строки — производные возрастающих порядков (от 0 до n−1).

Теорема 4.1 (основная теорема о вронскиане)

Пусть φ₁, ..., φₙ — решения ЛОДУ на (a, b). Тогда следующие три утверждения равносильны:

  1. {φ₁, ..., φₙ} — ФСР (система из n ЛНЗ решений)
  2. W(x) ≢ 0, т.е. существует хотя бы одна точка x₀ ∈ (a, b), где W(x₀) ≠ 0
  3. W(x) ≠ 0 для всех x ∈ (a, b)

Следствие: для решений ЛОДУ вронскиан либо тождественно равен нулю (ЛЗ), либо не обращается в ноль нигде на (a, b) (ЛНЗ). «Промежуточного» варианта нет.

Важно: это свойство «всё или ничего» верно ТОЛЬКО для решений ЛОДУ! Для произвольных функций вронскиан может обращаться в ноль в отдельных точках при ЛНЗ функциях.
Пример: проверка ЛНЗ

Функции: φ₁ = e^x, φ₂ = e^(2x). Вронскиан:

W(x) = |e^x e^(2x) | = e^x·2e^(2x) − e^(2x)·e^x = 2e^(3x) − e^(3x) = e^(3x)

|e^x 2e^(2x) |

W(x) = e^(3x) ≠ 0 при всех x → функции ЛНЗ.

Формула Лиувилля–Остроградского: W(x) = W(x₀)·exp(−∫_{x₀}^{x} aₙ₋₁(t) dt). Отсюда видно, что W либо всегда ≠ 0, либо тождественно 0.

ФСР. Существование ФСР. Структура общего решения (теоремы 4.2, 4.3)

Фундаментальная система решений (Определение 4.2)

ФСР — набор из n линейно независимых решений ЛОДУ n-го порядка: {φ₁, φ₂, ..., φₙ}.

Эквивалентно (по теореме 4.1): набор n решений, для которых W(φ₁, ..., φₙ) ≠ 0.

Теорема 4.2 (существование ФСР)

Утверждение: ФСР существует для любого ЛОДУ с непрерывными коэффициентами.

Доказательство (идея): Фиксируем точку x₀ ∈ (a, b). Решаем n задач Коши с начальными условиями, образующими единичную матрицу:

φ₁: y(x₀)=1, y'(x₀)=0, ..., y⁽ⁿ⁻¹⁾(x₀)=0 φ₂: y(x₀)=0, y'(x₀)=1, ..., y⁽ⁿ⁻¹⁾(x₀)=0 ... φₙ: y(x₀)=0, y'(x₀)=0, ..., y⁽ⁿ⁻¹⁾(x₀)=1

Каждая задача Коши имеет единственное решение (по теореме 3.2). Вычисляем вронскиан в точке x₀:

W(x₀) = det(E) = 1 ≠ 0

где E — единичная матрица (по построению). По теореме 4.1, W(x₀) ≠ 0 ⟹ φ₁, ..., φₙ — ЛНЗ ⟹ это ФСР.

Теорема 4.3 (структура общего решения ЛОДУ)

Утверждение: если {φ₁, ..., φₙ} — ФСР уравнения L[y] = 0, то:

y = C₁φ₁ + C₂φ₂ + ... + Cₙφₙ

является общим решением, т.е. любое решение ЛОДУ представимо в таком виде при подходящих C₁, ..., Cₙ.

Доказательство (идея): Пусть ψ(x) — произвольное решение. Покажем, что существуют C₁, ..., Cₙ такие, что ψ = C₁φ₁ + ... + Cₙφₙ. Приравниваем значения и производные в точке x₀:

C₁φ₁(x₀) + ... + Cₙφₙ(x₀) = ψ(x₀) C₁φ₁'(x₀) + ... + Cₙφₙ'(x₀) = ψ'(x₀) ... C₁φ₁⁽ⁿ⁻¹⁾(x₀) + ... + Cₙφₙ⁽ⁿ⁻¹⁾(x₀) = ψ⁽ⁿ⁻¹⁾(x₀)

Это система n линейных уравнений с определителем W(x₀) ≠ 0 → однозначно определяются C₁, ..., Cₙ. Функция y = C₁φ₁ + ... + Cₙφₙ — решение ЛОДУ с теми же начальными условиями, что и ψ. По теореме единственности: y ≡ ψ.

Для ЛНДУ (неоднородного)

Теорема 6.1. Общее решение ЛНДУ L[y] = q(x):

y = C₁φ₁ + C₂φ₂ + ... + Cₙφₙ + y*

где {φ₁, ..., φₙ} — ФСР однородного L[y] = 0, а y* — любое частное решение неоднородного уравнения.

Принцип: общее решение ЛНДУ = общее решение ЛОДУ + частное решение ЛНДУ. Сначала решаем однородное (находим ФСР), потом ищем одно частное неоднородного.
На экзамене: если просят «доказать теорему о структуре общего решения», нужно показать оба факта — (1) каждая линейная комбинация ФСР является решением; (2) каждое решение является такой комбинацией.

ЛОДУ с постоянными коэффициентами (теоремы 5.2, 5.3)

Вид уравнения (Параграф 5)

y⁽ⁿ⁾ + pₙ₋₁·y⁽ⁿ⁻¹⁾ + ... + p₁·y' + p₀·y = 0

Все коэффициенты p₀, p₁, ..., pₙ₋₁ — вещественные константы.

Характеристический многочлен (Определение 5.2)

Ищем решение в виде y = e^(λx). Подставляем: λⁿe^(λx) + pₙ₋₁λⁿ⁻¹e^(λx) + ... = 0. Сокращаем на e^(λx) ≠ 0:

D(λ) = λⁿ + pₙ₋₁·λⁿ⁻¹ + ... + p₁·λ + p₀ = 0

Это характеристическое уравнение. Его корни определяют ФСР.

Теорема 5.2 (основная теорема — построение ФСР)

Пусть λ₁, λ₂, ..., λₘ — различные корни характеристического многочлена D(λ) кратностей k₁, k₂, ..., kₘ (k₁ + k₂ + ... + kₘ = n). Тогда ФСР состоит из функций:

{x^r · e^(λⱼx)}, r = 0, 1, ..., kⱼ−1, j = 1, 2, ..., m

Каждому корню λⱼ кратности kⱼ соответствует kⱼ функций.

Разбор по случаям

Случай 1: вещественный простой корень λ (кратность 1):

→ e^(λx)

Случай 2: вещественный корень λ кратности r:

→ e^(λx), x·e^(λx), x²·e^(λx), ..., x^(r−1)·e^(λx)

(r функций в ФСР)

Теорема 5.3 (вещественная ФСР при вещественных коэффициентах)

Если все коэффициенты уравнения вещественные, то комплексные корни характеристического уравнения идут парами: α ± βi. Для каждой такой пары вместо комплексных экспонент берём вещественные функции:

Случай 3: комплексные корни α ± βi, кратность 1:

→ e^(αx)·cos(βx), e^(αx)·sin(βx)

(2 вещественные функции вместо 2 комплексных)

Случай 4: комплексные корни α ± βi, кратность r:

→ x^j·e^(αx)·cos(βx), x^j·e^(αx)·sin(βx) (j = 0, 1, ..., r−1)

(2r вещественных функций)

Пример 1: различные вещественные корни

y'' − 5y' + 6y = 0

Хар. ур-е: λ² − 5λ + 6 = 0 → λ₁ = 2, λ₂ = 3

ФСР: {e^(2x), e^(3x)}. Ответ: y = C₁e^(2x) + C₂e^(3x)

Пример 2: кратный корень

y'' − 4y' + 4y = 0

Хар. ур-е: λ² − 4λ + 4 = (λ − 2)² = 0 → λ = 2 кратности 2

ФСР: {e^(2x), x·e^(2x)}. Ответ: y = (C₁ + C₂x)·e^(2x)

Пример 3: комплексные корни

y'' + 2y' + 5y = 0

Хар. ур-е: λ² + 2λ + 5 = 0 → λ = −1 ± 2i (α = −1, β = 2)

ФСР: {e^(−x)cos 2x, e^(−x)sin 2x}. Ответ: y = e^(−x)(C₁cos 2x + C₂sin 2x)

Пример 4: 4-й порядок

y⁽⁴⁾ + 2y'' + y = 0

Хар. ур-е: λ⁴ + 2λ² + 1 = (λ² + 1)² = 0 → λ = ±i, кратность 2

α = 0, β = 1, r = 2. ФСР: {cos x, sin x, x·cos x, x·sin x}

Ответ: y = (C₁ + C₂x)cos x + (C₃ + C₄x)sin x

Алгоритм: (1) составить хар. уравнение, (2) найти все корни с кратностями, (3) по таблице выписать ФСР, (4) общее решение = линейная комбинация ФСР.

Метод Лагранжа (вариация постоянных) для ЛНДУ n-го порядка

Для чего нужен

Нахождение общего решения ЛНДУ: L[y] = q(x), т.е.:

y⁽ⁿ⁾ + aₙ₋₁(x)y⁽ⁿ⁻¹⁾ + ... + a₀(x)y = q(x)

Метод работает для любой правой части q(x) — в отличие от метода неопределённых коэффициентов!

Идея метода (Параграф 6)

Знаем ФСР однородного уравнения: {φ₁, ..., φₙ}. Общее решение ЛОДУ: y = C₁φ₁ + ... + Cₙφₙ. Заменяем постоянные C₁, ..., Cₙ на функции C₁(x), ..., Cₙ(x):

y = C₁(x)·φ₁ + C₂(x)·φ₂ + ... + Cₙ(x)·φₙ

Система для нахождения C₁'(x), ..., Cₙ'(x)

Чтобы y удовлетворяло неоднородному уравнению и чтобы производные y имели «правильный» вид (без производных Cₖ(x) в промежуточных формулах), накладываем n условий:

C₁'·φ₁ + C₂'·φ₂ + ... + Cₙ'·φₙ = 0 C₁'·φ₁' + C₂'·φ₂' + ... + Cₙ'·φₙ' = 0 ... C₁'·φ₁⁽ⁿ⁻²⁾ + C₂'·φ₂⁽ⁿ⁻²⁾ + ... + Cₙ'·φₙ⁽ⁿ⁻²⁾ = 0 C₁'·φ₁⁽ⁿ⁻¹⁾ + C₂'·φ₂⁽ⁿ⁻¹⁾ + ... + Cₙ'·φₙ⁽ⁿ⁻¹⁾ = q(x)

Первые (n−1) уравнений: условия, что промежуточные производные y имеют вид без C'. Последнее уравнение: подстановка в исходное ДУ.

Определитель этой системы = W(x) ≠ 0 (т.к. φ₁, ..., φₙ — ФСР). Значит, система имеет единственное решение для C₁', ..., Cₙ' (например, по формулам Крамера).

Алгоритм

  1. Решить ЛОДУ → найти ФСР {φ₁, ..., φₙ}
  2. Составить и решить систему для C₁', ..., Cₙ'
  3. Проинтегрировать: Cₖ(x) = ∫ Cₖ'(x) dx (без произвольных констант — они входят через общее решение ЛОДУ)
  4. Общее решение: y = C₁(x)φ₁ + ... + Cₙ(x)φₙ + D₁φ₁ + ... + Dₙφₙ (где Dₖ — произвольные константы)
Пример: y'' + y = 1/cos x

ЛОДУ: y'' + y = 0 → λ² + 1 = 0 → λ = ±i

ФСР: φ₁ = cos x, φ₂ = sin x

Система:

C₁'·cos x + C₂'·sin x = 0

−C₁'·sin x + C₂'·cos x = 1/cos x

Вронскиан: W = cos²x + sin²x = 1.

По Крамеру: C₁' = −sin x / cos x = −tan x, C₂' = 1.

Интегрируем: C₁ = ln|cos x|, C₂ = x.

Частное решение: y* = cos x · ln|cos x| + x · sin x

Общее: y = D₁cos x + D₂sin x + cos x · ln|cos x| + x · sin x

Метод Лагранжа универсален, но может приводить к сложным интегралам. Если правая часть имеет специальный вид (многочлен, экспонента, синус/косинус) — проще использовать метод неопределённых коэффициентов (билеты 16–17).

ЛНДУ: правая часть вида Pm(x)·eγx

Вид уравнения

ЛНДУ с постоянными коэффициентами:

y⁽ⁿ⁾ + pₙ₋₁y⁽ⁿ⁻¹⁾ + ... + p₀y = Pₘ(x) · e^(γx)

где Pₘ(x) = aₘxᵐ + aₘ₋₁xᵐ⁻¹ + ... + a₀ — многочлен степени m, γ — вещественное (или комплексное) число.

Вид частного решения (метод неопределённых коэффициентов)

y* = x^s · Qₘ(x) · e^(γx)
  • Qₘ(x) = bₘxᵐ + bₘ₋₁xᵐ⁻¹ + ... + b₀ — многочлен той же степени m с неизвестными коэффициентами
  • s — кратность числа γ как корня характеристического уравнения D(λ) = 0
  • Если γ НЕ является корнем характеристического уравнения → s = 0

Алгоритм

  1. Решаем характеристическое уравнение D(λ) = 0 → находим все корни
  2. Определяем s: является ли γ корнем D(λ) = 0? Если да — какой кратности?
  3. Записываем y* = x^s · Qₘ(x) · e^(γx) с неизвестными коэффициентами b₀, ..., bₘ
  4. Подставляем y* в уравнение, раскрываем и приравниваем коэффициенты при одинаковых степенях x
  5. Получаем систему для b₀, ..., bₘ → решаем → частное решение найдено
Почему работает множитель x^s? Если γ — корень D(λ) кратности s, то подстановка Qₘ·e^(γx) даст 0 (из-за кратности). Множитель x^s «сдвигает» на s степеней и делает систему разрешимой.
Пример 1: γ не корень (s = 0)

y'' − 3y' + 2y = e^(3x)

Хар. ур-е: λ² − 3λ + 2 = 0 → λ₁ = 1, λ₂ = 2. γ = 3 — не корень → s = 0

Подбор: y* = A·e^(3x) (m = 0, Q₀ = A)

Подставляем: 9A·e^(3x) − 9A·e^(3x) + 2A·e^(3x) = e^(3x) → 2A = 1 → A = 1/2

Общее: y = C₁e^x + C₂e^(2x) + e^(3x)/2

Пример 2: γ — простой корень (s = 1)

y'' − 3y' + 2y = e^(2x)

γ = 2 — корень кратности 1 → s = 1

Подбор: y* = x·A·e^(2x)

Подставляем, упрощаем: A(2e^(2x) + 4xe^(2x)) − 3A(e^(2x) + 2xe^(2x)) + 2A·x·e^(2x) = e^(2x)

Коэффициент при e^(2x): 2A − 3A = −A = 1 → A = −1

Ответ: y* = −x·e^(2x)

Пример 3: многочлен правая часть (γ = 0)

y'' − y = 2x² + 1

γ = 0, Pₘ(x) = 2x² + 1, m = 2. Хар. корни: λ = ±1. γ = 0 не корень → s = 0

Подбор: y* = ax² + bx + c

y*'' = 2a. Подставляем: 2a − (ax² + bx + c) = 2x² + 1

−a = 2 → a = −2; −b = 0 → b = 0; 2a − c = 1 → c = −5

y* = −2x² − 5

Главное — правильно определить s. Мнемоника: s = «сколько раз γ входит как корень». Если не входит — s = 0, подбираем «как есть».

ЛНДУ: правая часть вида eγx(Pncos βx + Qmsin βx)

Вид правой части

q(x) = e^(γx) · (Pₙ(x)·cos βx + Qₘ(x)·sin βx)

где Pₙ(x) — многочлен степени n, Qₘ(x) — многочлен степени m (один из них может быть нулевым).

Вид частного решения

y* = x^s · e^(γx) · (Rₗ(x)·cos βx + Tₗ(x)·sin βx)

где:

  • l = max(n, m) — степень многочленов Rₗ и Tₗ
  • Rₗ(x), Tₗ(x) — многочлены степени l с неизвестными коэффициентами
  • s — кратность числа γ + βi как корня характеристического уравнения
Оба многочлена Rₗ и Tₗ берутся степени l = max(n, m), даже если один из Pₙ, Qₘ отсутствует (нулевой). Это критически важно! Нельзя брать R и T разных степеней.

Связь с предыдущим билетом

Билет 16 — частный случай при β = 0 (нет тригонометрии). Формально: правая часть Pₘ·e^(γx) = e^(γx)·(Pₘ·cos 0x + 0·sin 0x), и γ + 0i = γ.

Алгоритм

  1. Определяем γ, β, n, m из правой части
  2. Составляем число γ + βi и проверяем: является ли оно корнем хар. уравнения? Если да — кратности s.
  3. Вычисляем l = max(n, m)
  4. Записываем y* = x^s · e^(γx) · (Rₗ·cos βx + Tₗ·sin βx)
  5. Подставляем в уравнение, приравниваем коэффициенты → находим все неизвестные коэффициенты Rₗ и Tₗ
Пример 1: y'' + y = cos x

Хар. ур-е: λ² + 1 = 0 → λ = ±i

Правая часть: e^(0x)·(1·cos x + 0·sin x) → γ = 0, β = 1, n = 0, m = 0

Число γ + βi = 0 + i = i — корень кратности 1 → s = 1

l = max(0, 0) = 0. Подбор: y* = x¹ · (A·cos x + B·sin x)

y* = x(A cos x + B sin x)

y*' = A cos x + B sin x + x(−A sin x + B cos x)

y*'' = −A sin x + B cos x + (−A sin x + B cos x) + x(−A cos x − B sin x)

y*'' = −2A sin x + 2B cos x − x(A cos x + B sin x)

y*'' + y* = −2A sin x + 2B cos x = cos x

Отсюда: B = 1/2, A = 0.

Ответ: y = C₁cos x + C₂sin x + (x sin x)/2

Пример 2: y'' − 2y' + y = e^x · sin x

Хар. ур-е: (λ − 1)² = 0 → λ = 1 (кратность 2)

Правая часть: e^(1·x)·(0·cos x + 1·sin x) → γ = 1, β = 1

γ + βi = 1 + i — НЕ корень хар. уравнения (корень только 1, а не 1+i) → s = 0

l = max(0, 0) = 0. Подбор: y* = e^x(A cos x + B sin x)

Подставляем, находим A и B.

Пример 3: y'' + 4y = x · sin 2x

Хар. ур-е: λ² + 4 = 0 → λ = ±2i

Правая часть: e^(0x)·(0·cos 2x + x·sin 2x) → γ = 0, β = 2, n = 0, m = 1

γ + βi = 2i — корень кратности 1 → s = 1

l = max(0, 1) = 1. Подбор: y* = x · ((ax + b)cos 2x + (cx + d)sin 2x)

Частый источник ошибок: неправильное определение s. Запомни: проверяем не γ, а число γ + βi! Например, если γ = 0 и β = 1, проверяем, является ли i корнем характеристического уравнения.

Векторная функция скалярного аргумента

Определение

Векторная функция скалярного аргумента — это функция, ставящая в соответствие каждому значению скалярного параметра t вектор:

r(t) = x(t)·i + y(t)·j + z(t)·k

где x(t), y(t), z(t) — скалярные функции (компоненты), i, j, k — единичные орты.

Годограф

Кривая в пространстве, которую описывает конец вектора r(t) при изменении параметра t. Если начало вектора зафиксировано в начале координат, годограф — это траектория движения точки.

Предел

Предел векторной функции берётся покомпонентно:

lim_{t→t₀} r(t) = (lim x(t))·i + (lim y(t))·j + (lim z(t))·k

Предел существует тогда и только тогда, когда существуют пределы каждой компоненты.

Непрерывность

r(t) непрерывна в t₀ ⟺ все компоненты x(t), y(t), z(t) непрерывны в t₀.

Производная

r'(t) = x'(t)·i + y'(t)·j + z'(t)·k

Геометрический смысл: r'(t) — вектор, касательный к годографу в точке r(t), направленный в сторону возрастания t.

Свойства дифференцирования

  • (r₁ ± r₂)' = r₁' ± r₂'
  • (φ(t)·r)' = φ'·r + φ·r' (скалярная функция на вектор)
  • (r₁ · r₂)' = r₁'·r₂ + r₁·r₂' (скалярное произведение)
  • (r₁ × r₂)' = r₁'×r₂ + r₁×r₂' (векторное произведение — порядок важен!)

Интеграл

∫ r(t) dt = (∫ x(t) dt)·i + (∫ y(t) dt)·j + (∫ z(t) dt)·k

Интегрирование тоже покомпонентное.

Все операции с векторной функцией выполняются покомпонентно. Это основа для записи систем ДУ в векторной форме (следующие билеты).

Системы ДУ: определения, задача Коши, сведение к одному ДУ

Определение нормальной системы (Определение 7.1)

Нормальная система ДУ — система, разрешённая относительно производных:

y₁' = f₁(x, y₁, y₂, ..., yₙ) y₂' = f₂(x, y₁, y₂, ..., yₙ) ... yₙ' = fₙ(x, y₁, y₂, ..., yₙ)

В векторной записи: Y'(x) = F(x, Y), где Y = (y₁, ..., yₙ)ᵀ.

Решение системы

  • Общее решение: yᵢ = φᵢ(x, C₁, ..., Cₙ) — n функций, содержащих n произвольных постоянных
  • Частное решение: при конкретных значениях C₁, ..., Cₙ

Задача Коши

Найти решение системы, удовлетворяющее начальным условиям:

y₁(x₀) = y₁⁰, y₂(x₀) = y₂⁰, ..., yₙ(x₀) = yₙ⁰

Теорема 7.1 (существования и единственности для системы)

Если функции fᵢ и все их частные производные ∂fᵢ/∂yⱼ непрерывны в окрестности начальной точки, то задача Коши имеет единственное решение.

Линейная система (Определение 7.3)

Система линейных ДУ (СЛДУ):

Y' = A(x)·Y + B(x)

где A(x) — матрица коэффициентов n×n, B(x) — вектор правых частей. Если B(x) ≡ 0 — однородная СЛДУ.

Сведение ДУ n-го порядка к системе

Из y⁽ⁿ⁾ = f(x, y, y', ..., y⁽ⁿ⁻¹⁾) вводим замену:

y₁ = y, y₂ = y', y₃ = y'', ..., yₙ = y⁽ⁿ⁻¹⁾

Получаем эквивалентную систему:

y₁' = y₂ y₂' = y₃ ... yₙ₋₁' = yₙ yₙ' = f(x, y₁, y₂, ..., yₙ)

Сведение системы к одному ДУ

Обратный процесс: из системы можно исключить все неизвестные, кроме одного, и получить одно ДУ высокого порядка. Метод: дифференцируем одно уравнение и подставляем другое, пока не останется одна функция.

Сведение ДУ ↔ система — это две стороны одной медали. Любое ДУ n-го порядка эквивалентно системе n уравнений 1-го порядка, и наоборот.

Методы решения линейных систем ДУ с постоянными коэффициентами

Вид системы

X' = A · X

где A — матрица постоянных коэффициентов размера n×n, X = (x₁, ..., xₙ)ᵀ — вектор неизвестных функций.

Метод 1: Метод исключения

Сводим систему к одному ДУ высокого порядка:

  1. Из одного уравнения выражаем y₂ через y₁ и y₁'
  2. Дифференцируем это выражение
  3. Подставляем во второе уравнение → получаем одно ДУ n-го порядка для y₁
  4. Решаем это ДУ → находим y₁ → обратно подставляем → y₂
При методе исключения возникают «лишние» постоянные. Не все C₁, ..., Cₙ независимы — часть из них связана соотношениями из исходной системы. Нужно проверить!

Метод 2: Метод собственных значений (основной)

Ищем решение в виде X = h · e^(λx), где h — постоянный вектор, λ — число.

Подставляем в X' = AX: λh·e^(λx) = A·h·e^(λx) → (A − λE)·h = 0

Это задача на собственные значения матрицы A.

  1. Характеристическое уравнение: det(A − λE) = 0 → находим собственные числа λ₁, ..., λₙ
  2. Для каждого λᵢ находим собственный вектор hᵢ: решаем (A − λᵢE)·hᵢ = 0
  3. Каждая пара (λᵢ, hᵢ) даёт частное решение: Xᵢ = hᵢ · e^(λᵢx)

Общее решение (случай различных собственных значений)

X = C₁·h₁·e^(λ₁x) + C₂·h₂·e^(λ₂x) + ... + Cₙ·hₙ·e^(λₙx)

Особый случай: кратные собственные значения

Если λ — корень кратности r, а геометрическая кратность (число ЛНЗ собственных векторов) меньше r, используем присоединённые (обобщённые) собственные вектора:

Для кратности 2 с одним собственным вектором h₁ ищем h₂ из (A − λE)·h₂ = h₁.

Второе частное решение:

X₂ = (h₁·x + h₂) · e^(λx)

Особый случай: комплексные собственные значения

При вещественной матрице A комплексные корни идут парами: λ = α ± βi. Пусть h = u + iv — собственный вектор для λ = α + βi. Тогда два вещественных решения:

X₁ = e^(αx)·(u·cos βx − v·sin βx) X₂ = e^(αx)·(u·sin βx + v·cos βx)

Это действительная и мнимая части комплексного решения (u + iv)·e^((α+βi)x).

Пример: различные вещественные корни

Система: x' = 3x + y, y' = x + 3y

Матрица A = |3 1| / |1 3|

det(A − λE) = (3−λ)² − 1 = λ² − 6λ + 8 = 0 → λ₁ = 2, λ₂ = 4

λ₁ = 2: (A − 2E)h = 0 → |1 1|·h = 0 → h₁ = (1, −1)ᵀ

λ₂ = 4: (A − 4E)h = 0 → |−1 1|·h = 0 → h₂ = (1, 1)ᵀ

Ответ:

x = C₁e^(2t) + C₂e^(4t) y = −C₁e^(2t) + C₂e^(4t)
Пример: комплексные корни

Система: x' = −y, y' = x. Матрица A = |0 −1| / |1 0|

det(A − λE) = λ² + 1 = 0 → λ = ±i (α = 0, β = 1)

Для λ = i: (A − iE)h = 0 → h = (1, i)ᵀ = (1, 0)ᵀ + i·(0, 1)ᵀ

u = (1, 0)ᵀ, v = (0, 1)ᵀ

Ответ: x = C₁cos t + C₂sin t, y = −C₁sin t + C₂cos t (вращение!)

Неоднородная система X' = AX + B(x)

Общее решение = общее решение однородной + частное неоднородной. Частное можно искать методом вариации постоянных (аналог метода Лагранжа для систем).

Метод собственных значений — основной на экзамене. Алгоритм: (1) хар. уравнение, (2) собственные значения, (3) собственные вектора, (4) записать общее решение.
