Понятие числового ряда. Сходимость. Необходимое условие

Определение 1.1. Числовой ряд

Числовой ряд -- формальная бесконечная сумма членов числовой последовательности {an}:

∑ aₙ = a₁ + a₂ + a₃ + ... + aₙ + ... (n = 1, 2, 3, ...)

Здесь an -- общий (n-й) член ряда.

Частичная сумма

n-я частичная сумма -- сумма первых n членов ряда:

Sₙ = a₁ + a₂ + ... + aₙ = ∑(k=1..n) aₖ

Последовательность {Sn} -- ключевой объект для изучения сходимости.

Определение сходимости

Ряд сходится, если существует конечный предел последовательности частичных сумм:

S = lim(n→∞) Sₙ (конечный)

Число S называется суммой ряда. Если предел не существует или равен ±∞ -- ряд расходится.

Лемма 1.1. Гармонический ряд

Для частичных сумм гармонического ряда ∑1/n справедлива асимптотическая формула:

Sₙ = 1 + 1/2 + 1/3 + ... + 1/n = c + ln n + εₙ

где c = 0.5772... -- постоянная Эйлера (Euler-Mascheroni constant), εₙ → 0 при n → ∞.

Следствие: Sn → ∞ (т.к. ln n → ∞), поэтому гармонический ряд расходится.

Идея доказательства расходимости (без формулы): группируем члены: (1) + (1/2) + (1/3+1/4) + (1/5+...+1/8) + ... Каждая группа ≥ 1/2, а таких групп бесконечно много.

Определение 1.2. Остаток ряда

rₘ = aₘ₊₁ + aₘ₊₂ + ... = ∑(n=m+1..∞) aₙ = S − Sₘ

Для сходящегося ряда: lim(m→∞) rₘ = 0.

Теорема 1.3. Ряд и его остаток

Ряд ∑an и его остаток rm = ∑(n=m+1..∞) an сходятся или расходятся одновременно.

Суть: отбрасывание конечного числа первых членов не влияет на сходимость (влияет только на сумму).

Теорема 1.4. Необходимый признак сходимости

Если ∑aₙ сходится, то lim(n→∞) aₙ = 0

Доказательство: an = Sn − Sn−1. Если ∑an сходится, то lim Sn = S и lim Sn−1 = S. Значит lim an = S − S = 0.

Обратное НЕВЕРНО! lim aₙ = 0 НЕ гарантирует сходимость. Гармонический ряд ∑1/n: lim(1/n) = 0, но ряд расходится. Это типичная экзаменационная ловушка!
Следствие (контрапозиция): если lim aₙ ≠ 0 или предел не существует, то ряд точно расходится. Это самая быстрая первая проверка на экзамене.

Свойства сходящихся числовых рядов

Теорема 1.2. Линейность

Если ∑an = A и ∑bn = B (оба сходятся), то для любых констант α, β:

∑(α·aₙ + β·bₙ) = α·A + β·B

Доказательство: частичная сумма нового ряда: σn = α·Sn(a) + β·Sn(b). Переходим к пределу: lim σn = α·A + β·B.

Теорема 1.1. Группировка членов

Если ряд ∑an сходится, то его члены можно группировать в скобки (не переставляя!) -- новый ряд тоже сходится и имеет ту же сумму.

Идея доказательства: частичные суммы нового ряда -- подпоследовательность {Sn}, а подпоследовательность сходящейся последовательности сходится к тому же пределу.

Обратное НЕВЕРНО! Из сходимости ряда со скобками не следует сходимость без скобок. Пример: (1−1)+(1−1)+... = 0 сходится, но 1−1+1−1+... расходится.

Отбрасывание/добавление конечного числа членов

Сходимость ряда не зависит от конечного числа первых членов (это следует из теоремы 1.3 об остатке).

Сходящийся + расходящийся

Если ∑an сходится и ∑bn расходится, то ∑(an + bn) расходится.

Доказательство от противного: если бы ∑(an + bn) сходился, то ∑bn = ∑(an+bn) − ∑an тоже сходился бы (линейность). Противоречие.

Сумма двух расходящихся рядов может быть любой! Пример: aₙ = 1/n и bₙ = −1/n -- оба расходятся, но ∑(aₙ + bₙ) = 0.

Признаки сравнения для рядов с неотрицательными членами

Теорема 2.1. Критерий сходимости

Ряд с неотрицательными членами (an ≥ 0) сходится тогда и только тогда, когда последовательность частичных сумм {Sn} ограничена сверху.

Доказательство: {Sn} монотонно не убывает (т.к. Sn+1 = Sn + an+1 ≥ Sn). По теореме Вейерштрасса монотонная ограниченная последовательность имеет предел.

Теорема 2.2. Первый признак сравнения

Пусть 0 ≤ aₙ ≤ bₙ для всех n (начиная с некоторого N):

∑bₙ сходится → ∑aₙ сходится ∑aₙ расходится → ∑bₙ расходится

Доказательство (первая часть): Sn(a) ≤ Sn(b) ≤ B (ограничена). Значит {Sn(a)} ограничена сверху, а по Теореме 2.1 ряд ∑an сходится.

Мнемоника: меньшее сходится с большим, большее расходится с меньшим.

Теорема 2.2 (продолжение). Второй признак сравнения (предельный)

lim(n→∞) (aₙ / bₙ) = L, 0 < L < ∞ → ряды ведут себя одинаково

Оба сходятся или оба расходятся.

Доказательство: из определения предела: при больших n выполнено (L/2)·bn ≤ an ≤ (3L/2)·bn. Дальше применяем первый признак сравнения.

Частные случаи:

  • L = 0: сходимость ∑bn → сходимость ∑an (an «мала» по сравнению с bn)
  • L = ∞: расходимость ∑bn → расходимость ∑an
Пример

Исследовать ∑ 1/(n² + n)

Сравниваем с эталоном ∑ 1/n² (сходится, p = 2 > 1):

lim [1/(n²+n)] / [1/n²] = lim n²/(n²+n) = lim 1/(1+1/n) = 1

0 < 1 < ∞ → ряд сходится.

Эталонные ряды, которые нужно знать наизусть: (1) Обобщённый гармонический ∑1/nᵖ: сходится при p > 1, расходится при p ≤ 1. (2) Геометрический ∑qⁿ: сходится при |q| < 1 (сумма = 1/(1−q)).

Признаки Даламбера, Коши, интегральный. Признак Раабе. Оценка остатка

Теоремы 2.3-2.4. Признак Даламбера

Пусть an > 0. Рассмотрим предел:

q = lim(n→∞) (aₙ₊₁ / aₙ)
  • q < 1 → ряд сходится (абсолютно)
  • q > 1 → ряд расходится (аn → ∞)
  • q = 1 → признак не работает (нужны другие методы)

Идея доказательства (q < 1): выбираем r: q < r < 1. Начиная с некоторого N: an+1/an < r, т.е. an+1 < r·an. По индукции: aN+k < rk·aN. Сравниваем с геометрической прогрессией ∑rk, которая сходится при r < 1.

Теоремы 2.5-2.6. Радикальный признак Коши

q = lim(n→∞) ⁿ√aₙ = lim aₙ^(1/n)
  • q < 1 → ряд сходится
  • q > 1 → ряд расходится
  • q = 1 → признак не работает

Идея доказательства: если q < 1, берём r: q < r < 1. Тогда при больших n: an1/n < r, т.е. an < rn. Сравнение с ∑rn.

Признак Коши «сильнее» Даламбера: если Даламбер даёт ответ, то Коши тоже, но не наоборот. Однако на практике Даламбер удобнее вычислять.

Теорема 2.7. Интегральный признак Коши-Маклорена

Пусть f(x) ≥ 0, непрерывна и монотонно убывает на [1, ∞), и an = f(n). Тогда:

∑(n=1..∞) f(n) и ∫₁^∞ f(x)dx сходятся или расходятся одновременно

Идея доказательства: f убывает, поэтому f(n+1) ≤ ∫nn+1 f(x)dx ≤ f(n). Суммируем: Sn − a1 ≤ ∫1n+1 f(x)dx ≤ Sn.

Оценка остатка (через интеграл)

∫ₙ₊₁^∞ f(x)dx ≤ rₙ ≤ ∫ₙ^∞ f(x)dx

Следствие. Обобщённый гармонический ряд

∑ 1/nᵖ : сходится при p > 1, расходится при p ≤ 1

Доказательство: f(x) = 1/xp удовлетворяет условиям. ∫1∞ dx/xp = [x1−p/(1−p)]1∞. При p > 1: сходится (= 1/(p−1)). При p < 1: расходится. При p = 1: ∫dx/x = ln x → ∞.

Признак Раабе

Если признак Даламбера даёт q = 1 (неопределённость), применяем признак Раабе:

R = lim(n→∞) n · (aₙ/aₙ₊₁ − 1)
  • R > 1 → ряд сходится
  • R < 1 → ряд расходится
  • R = 1 → признак не работает
Пример (Даламбер)

∑ n!/nⁿ

aₙ₊₁/aₙ = (n+1)!·nⁿ / ((n+1)ⁿ⁺¹·n!) = nⁿ/(n+1)ⁿ = (n/(n+1))ⁿ = (1 − 1/(n+1))ⁿ → 1/e

q = 1/e < 1 → ряд сходится.

Когда какой признак

Д'Аламбер: ряды с факториалами, степенями n. Пример: ∑n!/3ⁿ, ∑nⁿ/n!.

Коши (радикальный): ряды вида ∑ann. Пример: ∑(n/(2n+1))ⁿ.

Интегральный: когда f(n) легко интегрируется. Пример: ∑1/(n·ln²n).

Раабе: когда Даламбер даёт 1. Пример: ∑1·3·5·...·(2n−1) / (2·4·6·...·(2n)).

Знакочередующиеся ряды. Признак Лейбница. Оценка остатка

Определение

Знакочередующийся ряд -- ряд, в котором знаки строго чередуются:

∑ (−1)ⁿ⁺¹ · bₙ = b₁ − b₂ + b₃ − b₄ + ... (bₙ > 0)

Теорема 3.3. Признак Лейбница

Знакочередующийся ряд ∑(−1)n+1bn (bn > 0) сходится, если выполнены два условия:

  1. bₙ₊₁ ≤ bₙ -- члены монотонно убывают по модулю (bn↓)
  2. lim(n→∞) bₙ = 0

Доказательство сходимости

Шаг 1. Рассматриваем S2n (чётные частичные суммы):

S₂ₙ = (b₁ − b₂) + (b₃ − b₄) + ... + (b₂ₙ₋₁ − b₂ₙ)

Каждая скобка ≥ 0 (т.к. bk↓), значит S2n монотонно не убывает.

Шаг 2. Перегруппировка:

S₂ₙ = b₁ − (b₂ − b₃) − (b₄ − b₅) − ... − b₂ₙ ≤ b₁

Значит S2n ограничена сверху. По теореме Вейерштрасса ∃ lim S2n = S.

Шаг 3. S2n+1 = S2n + b2n+1 → S + 0 = S.

Итого: lim Sn = S -- ряд сходится.

Оценка остатка

|rₘ| = |S − Sₘ| ≤ bₘ₊₁

Погрешность не превышает первого отброшенного члена (по модулю).

Доказательство оценки:

Остаток: rm = bm+1 − bm+2 + bm+3 − ...

Группируем: rm = (bm+1 − bm+2) + (bm+3 − bm+4) + ... ≥ 0

Группируем иначе: rm = bm+1 − (bm+2 − bm+3) − ... ≤ bm+1

Итого: 0 ≤ rₘ ≤ bₘ₊₁

Пример

∑ (−1)ⁿ⁺¹/n = 1 − 1/2 + 1/3 − 1/4 + ...

bn = 1/n: убывает, lim = 0 → сходится (сумма = ln 2).

Берём S₃ = 1 − 1/2 + 1/3 = 5/6: погрешность |r₃| ≤ b₄ = 1/4 = 0.25.

Знакопеременные ряды. Абсолютная и условная сходимость

Знакопеременный ряд

Ряд ∑an, члены которого имеют произвольные знаки (не обязательно чередуются). Знакочередующийся ряд -- частный случай.

Теорема 3.1. Абсолютная сходимость → сходимость

∑|aₙ| сходится → ∑aₙ сходится

Доказательство: Пусть ∑|an| сходится. По критерию Больцано-Коши для ∑|an|: для ∀ε > 0 ∃N: |an+1| + ... + |an+p| < ε при n > N. Но по неравенству треугольника: |an+1 + ... + an+p| ≤ |an+1| + ... + |an+p| < ε. Значит критерий Больцано-Коши выполнен и для ∑an.

Определение 3.1. Типы сходимости

Абсолютная сходимость: ∑aₙ сходится и ∑|aₙ| сходится Условная сходимость: ∑aₙ сходится, но ∑|aₙ| расходится
Примеры

∑ (−1)ⁿ⁺¹/n² — абсолютно сходится (т.к. ∑1/n² сходится, p=2 > 1)

∑ (−1)ⁿ⁺¹/n — условно сходится (сам сходится по Лейбницу, но ∑1/n расходится)

Теорема 3.4. Произведение рядов (по Коши)

Если ∑an = A и ∑bn = B сходятся абсолютно, то их произведение по Коши:

c₁ = a₁b₁, c₂ = a₁b₂+a₂b₁, c₃ = a₁b₃+a₂b₂+a₃b₁, ... cₙ = ∑(k=1..n) aₖ·bₙ₋ₖ₊₁

сходится абсолютно и ∑cn = A·B.

Теорема Римана (о перестановке)

Если ряд сходится условно, то перестановкой его членов можно получить ряд с любой наперёд заданной суммой (или расходящийся ряд).

Вот почему абсолютная сходимость важна: в абсолютно сходящихся рядах перестановка членов не меняет сумму. В условно сходящихся -- сумма зависит от порядка!

Признаки сходимости знакопеременных рядов. Дирихле, Абель

Основной метод: через абсолютную сходимость

∑|aₙ| сходится → ∑aₙ сходится (абсолютно)

Сначала исследуем ∑|an| признаками для рядов с положительными членами (Даламбер, Коши, сравнение, интегральный). Если он сходится -- готово.

Теорема 3.2. Признак Дирихле

Ряд ∑ aₙ·bₙ сходится, если:

  1. Частичные суммы ∑bn ограничены: |Bn| = |b₁ + b₂ + ... + bₙ| ≤ M для всех n
  2. Последовательность {an} монотонно стремится к нулю: an↓0 (или an↑0)

Следствие. Признак Абеля

Ряд ∑ aₙ·bₙ сходится, если:

  1. Ряд ∑bn сходится
  2. Последовательность {an} монотонна и ограничена

Почему это следствие Дирихле: если ∑bn сходится, то его частичные суммы автоматически ограничены. Монотонная ограниченная последовательность имеет предел c, и an − c → 0 монотонно. Разбиваем: ∑anbn = ∑(an−c)bn + c·∑bn.

Связь Лейбница и Дирихле

Признак Лейбница -- частный случай Дирихле при bn = (−1)n:

Частичные суммы ∑(−1)k ограничены (колеблются между −1 и 0), а an = bn монотонно стремится к 0.

Критерий Больцано-Коши для числовых рядов

Ряд ∑an сходится ⟺ для ∀ε > 0 ∃N: для всех n > N и ∀p ≥ 1:

|aₙ₊₁ + aₙ₊₂ + ... + aₙ₊ₚ| < ε

Любая «хвостовая» сумма сколь угодно мала. Этот критерий используется при доказательстве теорем (Дирихле, Абеля), но редко для конкретных рядов.

Алгоритм на экзамене: (1) Проверить необходимый признак lim aₙ = 0. (2) Проверить абсолютную сходимость через ∑|aₙ|. (3) Если ∑|aₙ| расходится -- попробовать Лейбниц/Дирихле/Абель для условной сходимости.
Функциональные ряды (Лекции 10-11)

Функциональный ряд. Область сходимости. Равномерная сходимость

Определения 4.1-4.2

Функциональный ряд -- ряд, членами которого являются функции:

∑ uₙ(x) = u₁(x) + u₂(x) + u₃(x) + ...

При каждом фиксированном x0 получаем обычный числовой ряд ∑un(x0).

Область сходимости Ω -- множество всех x, при которых числовой ряд сходится.

Сумма ряда: S(x) = lim Sn(x), x ∈ Ω.

Поточечная сходимость

∀x ∈ Ω, ∀ε > 0 ∃N = N(ε, x): при n > N выполнено |S(x) − Sn(x)| < ε.

N зависит от x -- для разных точек может потребоваться разное число членов.

Определение 4.3. Равномерная сходимость

∑un(x) сходится равномерно на Ω, если:

∀ε > 0 ∃N (НЕ зависящий от x): ∀n > N, ∀x ∈ Ω: |S(x) − Sₙ(x)| < ε

Ключевое отличие: одно и то же N работает для всех x сразу!

Обозначение: ∑un(x) ⇒ S(x) на Ω (двойная стрелка).

Теорема 4.1. Необходимый признак

Если ∑un(x) ⇒ на Ω, то un(x) ⇒ 0 на Ω (равномерно стремится к нулю).

Геометрическая интерпретация

Равномерная сходимость означает, что графики Sn(x) «равномерно приближаются» к S(x) -- при больших n весь график Sn(x) лежит в ε-полосе вокруг S(x).

Равномерная сходимость -- ключевое условие для переноса свойств (непрерывность, интегрируемость, дифференцируемость) с членов ряда на его сумму. Без неё эти свойства могут нарушаться.

Критерий Больцано-Коши. Признак Вейерштрасса

Критерий Больцано-Коши (равномерная сходимость)

∑un(x) сходится равномерно на X ⟺

∀ε > 0 ∃N: для всех n > N, ∀p ≥ 1, ∀x ∈ X:

|uₙ₊₁(x) + uₙ₊₂(x) + ... + uₙ₊ₚ(x)| < ε

Аналог числового критерия, но с требованием одновременно для всех x.

Критерий Больцано-Коши полезен для доказательства неравномерной сходимости: достаточно предъявить последовательности xn, nk, pk, при которых частичная сумма не стремится к нулю.

Теорема 4.3. Признак Вейерштрасса (мажорантный)

Если существует сходящийся числовой ряд ∑cn (cn ≥ 0) такой, что:

|uₙ(x)| ≤ cₙ для всех x ∈ X и всех n ≥ 1

то ∑un(x) сходится равномерно и абсолютно на X.

Ряд ∑cn называется мажорантным рядом (мажорантой).

Доказательство: по критерию Больцано-Коши: |un+1(x) + ... + un+p(x)| ≤ |un+1(x)| + ... + |un+p(x)| ≤ cn+1 + ... + cn+p < ε (т.к. ∑cn сходится). Причём N не зависит от x.

Пример

Ряд ∑ sin(nx)/n²

|sin(nx)/n²| ≤ 1/n² для всех x

∑1/n² сходится (p = 2 > 1) → по Вейерштрассу ряд сходится равномерно на всей R.

Пример: когда Вейерштрасс не работает

Ряд ∑ (−1)ⁿ·xⁿ/n на (−1, 1).

Попытка: |xn/n| ≤ 1/n -- но ∑1/n расходится! Вейерштрасс не применим.

Ряд всё же сходится равномерно на [−r, r] при r < 1 (мажоранта ∑rⁿ/n сходится).

Признак Вейерштрасса -- самый простой и частый способ доказать равномерную сходимость. Найди мажоранту -- числовой ряд, которым можно «зажать» функциональный!

Непрерывность суммы функционального ряда

Теорема 5.1 (о непрерывности суммы)

Если:

  • Все un(x) непрерывны на [a, b]
  • ∑un(x) сходится равномерно на [a, b]

то сумма S(x) = ∑un(x) тоже непрерывна на [a, b].

uₙ непрерывны + равномерная сходимость → S(x) непрерывна

Эквивалентная формулировка (почленный предельный переход):

lim(x→x₀) ∑uₙ(x) = ∑ lim(x→x₀) uₙ(x) = ∑uₙ(x₀)

Можно менять местами знак предела и знак суммы ряда.

Идея доказательства

|S(x) − S(x₀)| ≤ |S(x) − Sn(x)| + |Sn(x) − Sn(x₀)| + |Sn(x₀) − S(x₀)|

Первое и третье слагаемое < ε/3 (равномерная сходимость, N не зависит от x). Второе < ε/3 (непрерывность Sn как конечной суммы непрерывных).

Без равномерной сходимости непрерывность может нарушаться! Классический пример: ∑(xⁿ − xⁿ⁻¹) на [0,1] сходится поточечно к разрывной функции (0 при 0 ≤ x < 1, −1 при x = 1), хотя каждый член непрерывен.
Следствие: если сумма ряда непрерывных функций разрывна, то сходимость точно неравномерна. Это удобный способ опровергнуть равномерную сходимость.

Почленное интегрирование функционального ряда

Теорема 5.2

Если:

  • Все un(x) непрерывны на [a, b]
  • ∑un(x) сходится равномерно на [a, b]

то ряд можно интегрировать почленно:

∫ₐᵇ S(x)dx = ∫ₐᵇ [∑uₙ(x)] dx = ∑ ∫ₐᵇ uₙ(x) dx

Знак суммы ряда и знак интеграла можно менять местами.

Идея доказательства

|∫ab S(x)dx − ∫ab Sn(x)dx| = |∫ab [S(x) − Sn(x)]dx| ≤ ∫ab |S(x) − Sn(x)|dx ≤ ε·(b−a)

Последнее неравенство -- из определения равномерной сходимости (|S − Sn| < ε для всех x).

На практике: если доказана равномерная сходимость (например, по Вейерштрассу), можно смело интегрировать каждый член отдельно и сложить результаты.
Интегрирование «сглаживает», поэтому условия для почленного интегрирования мягче, чем для дифференцирования. Достаточно равномерной сходимости самого ряда.

Почленное дифференцирование функционального ряда

Теорема 5.3

Если:

  1. Все un(x) имеют непрерывные производные un'(x) на [a, b]
  2. Ряд ∑un(x) сходится хотя бы в одной точке x₀ ∈ [a, b]
  3. Ряд производных ∑un'(x) сходится равномерно на [a, b]

то:

S'(x) = (∑uₙ(x))' = ∑ uₙ'(x) на [a, b]

Причём ∑un(x) тоже сходится равномерно на [a, b].

Идея доказательства

Положим φ(x) = ∑un'(x) (сходится равномерно по условию 3). По теореме о почленном интегрировании:

∫ₓ₀ˣ φ(t)dt = ∑ ∫ₓ₀ˣ uₙ'(t)dt = ∑ [uₙ(x) − uₙ(x₀)] = S(x) − S(x₀)

Дифференцируя: S'(x) = φ(x) = ∑un'(x).

Ключевое отличие от интегрирования: здесь нужна равномерная сходимость ряда ПРОИЗВОДНЫХ ∑uₙ'(x), а не самого ряда! Это более жёсткое условие.
Мнемоника: интегрирование «сглаживает» (мягкие условия), дифференцирование «обостряет» (жёсткие условия). Для дифференцирования нужно контролировать производные.
Степенные ряды (Лекция 13)

Степенной ряд. Теорема Абеля. Радиус сходимости

Определение

∑(n=0..∞) cₙ(x − x₀)ⁿ = c₀ + c₁(x−x₀) + c₂(x−x₀)² + ...

x₀ -- центр ряда, cn -- коэффициенты. При x₀ = 0: ∑cnxn.

Теорема 6.1 (Абеля). Структура области сходимости

Часть 1. Если степенной ряд сходится при x = x₁ ≠ x₀, то он сходится абсолютно для всех x таких, что |x − x₀| < |x₁ − x₀|.

Часть 2. Если ряд расходится при x = x₂, то он расходится для всех x: |x − x₀| > |x₂ − x₀|.

Доказательство (часть 1)

Ряд сходится при x₁ → cn(x₁−x₀)n → 0 → |cn(x₁−x₀)n| ≤ M для всех n.

Пусть |x−x₀| < |x₁−x₀|. Обозначим q = |x−x₀|/|x₁−x₀| < 1.

|cₙ(x−x₀)ⁿ| = |cₙ(x₁−x₀)ⁿ| · |(x−x₀)/(x₁−x₀)|ⁿ ≤ M·qⁿ

Ряд ∑M·qn -- сходящаяся геометрическая прогрессия (q < 1). По признаку сравнения ∑|cn(x−x₀)n| сходится.

Радиус сходимости R

Существует число R ∈ [0, +∞] такое, что:

  • |x − x₀| < R → ряд сходится абсолютно
  • |x − x₀| > R → ряд расходится
  • |x − x₀| = R → нужно проверять отдельно (может быть и так, и так)

Интервал (x₀ − R, x₀ + R) -- интервал сходимости. Область сходимости = интервал + проверка концов.

Формулы для вычисления R

R = lim(n→∞) |cₙ / cₙ₊₁| (формула Даламбера, если предел существует)
1/R = lim sup(n→∞) |cₙ|^(1/n) (формула Коши-Адамара, всегда работает)

Соглашения: R = ∞ означает сходимость при всех x; R = 0 означает сходимость только при x = x₀.

Равномерная сходимость

Степенной ряд сходится равномерно на любом отрезке [x₀−r, x₀+r], где 0 < r < R.

Доказательство: |cn(x−x₀)n| ≤ |cn|·rn, а ∑|cn|rn сходится (т.к. r < R). По Вейерштрассу.

Пример

∑ xⁿ/n!

|cₙ/cₙ₊₁| = |1/n!| / |1/(n+1)!| = (n+1)! / n! = n+1 → ∞

R = ∞ -- ряд сходится при всех x (это ex).

Свойства суммы степенного ряда. Теорема Абеля о непрерывности на конце

Теорема 6.3. Почленное интегрирование степенного ряда

Если S(x) = ∑cn(x−x₀)n с радиусом R, то:

∫ₓ₀ˣ S(t)dt = ∑ cₙ(x−x₀)ⁿ⁺¹/(n+1)

Радиус сходимости не меняется (остаётся R).

Теорема 6.4. Почленное дифференцирование степенного ряда

S'(x) = ∑ n·cₙ(x−x₀)ⁿ⁻¹

Радиус сходимости не меняется (остаётся R).

Доказательство сохранения R при дифференцировании: для ряда ∑n·cn·tn−1 применим Коши-Адамара: lim sup (n|cn|)1/n = lim sup |cn|1/n · lim n1/n = 1/R · 1 = 1/R. Значит R' = R.

Степенные ряды можно дифференцировать и интегрировать почленно сколько угодно раз! Радиус R при этом сохраняется. Может измениться только поведение на концах интервала.

Непрерывность суммы

Сумма S(x) = ∑cn(x−x₀)n непрерывна на (x₀−R, x₀+R). Это следует из равномерной сходимости на каждом [x₀−r, x₀+r] (r < R) и теоремы 5.1.

Теорема 6.2 (Абеля, о непрерывности на конце)

Если степенной ряд ∑cnxn сходится при x = R (на правом конце интервала), то:

lim(x→R⁻) S(x) = S(R) = ∑ cₙRⁿ

То есть сумма ряда непрерывна слева в точке x = R. Аналогично для левого конца x = −R (непрерывна справа).

Применение теоремы Абеля

Ряд ∑(−1)ⁿ⁺¹xⁿ/n имеет R = 1. При x = 1 получаем ∑(−1)n+1/n = ln 2 (сходится по Лейбницу).

Сумма ряда: S(x) = ln(1+x) на (−1, 1).

По теореме Абеля: lim(x→1⁻) ln(1+x) = S(1), т.е. ln 2 = ∑(−1)n+1/n.

Ряд Тейлора. Ряд Маклорена. Достаточное условие разложимости

Ряд Тейлора

Если f(x) бесконечно дифференцируема в окрестности точки x₀, то ей можно формально сопоставить степенной ряд:

∑(n=0..∞) f⁽ⁿ⁾(x₀)/n! · (x − x₀)ⁿ

Ряд Маклорена

Частный случай при x₀ = 0:

f(x) = f(0) + f'(0)·x + f''(0)·x²/2! + f'''(0)·x³/3! + ...

Формула Тейлора с остатком

f(x) = Sₙ(x) + Rₙ(x)

где Sn(x) = ∑(k=0..n) f(k)(x₀)/k! · (x−x₀)k -- многочлен Тейлора степени n.

Остаток в форме Лагранжа:

Rₙ(x) = f⁽ⁿ⁺¹⁾(ξ)/(n+1)! · (x − x₀)ⁿ⁺¹, где ξ между x₀ и x

Критерий разложимости

f(x) = ∑ f⁽ⁿ⁾(x₀)/n! · (x−x₀)ⁿ ⟺ lim(n→∞) Rₙ(x) = 0

Ряд Тейлора сходится к f(x), если и только если остаток стремится к нулю.

Теорема 6.5. Достаточное условие

Если существует M > 0 такое, что |f⁽ⁿ⁾(x)| ≤ M для всех n и всех x из некоторой окрестности x₀, то ряд Тейлора сходится к f(x).

Доказательство: используем остаток Лагранжа. Пусть |x − x₀| ≤ H. Тогда:

|Rₙ(x)| ≤ M · Hⁿ⁺¹/(n+1)! → 0 при n → ∞

Потому что факториал растёт быстрее любой степени: Hn/n! → 0 для любого H.

Не всякая бесконечно дифференцируемая функция разложима в ряд Тейлора! Контрпример: f(x) = e−1/x² при x≠0, f(0) = 0. Все производные в нуле равны 0, ряд Маклорена = 0, но f(x) ≠ 0 при x ≠ 0.

Разложение основных элементарных функций в ряд Маклорена

Таблица разложений (знать наизусть!)

eˣ = ∑(n=0..∞) xⁿ/n! = 1 + x + x²/2! + x³/3! + ... R = ∞
sin x = ∑(n=0..∞) (−1)ⁿ · x²ⁿ⁺¹/(2n+1)! = x − x³/3! + x⁵/5! − ... R = ∞
cos x = ∑(n=0..∞) (−1)ⁿ · x²ⁿ/(2n)! = 1 − x²/2! + x⁴/4! − ... R = ∞
ch x = ∑(n=0..∞) x²ⁿ/(2n)! = 1 + x²/2! + x⁴/4! + ... R = ∞
sh x = ∑(n=0..∞) x²ⁿ⁺¹/(2n+1)! = x + x³/3! + x⁵/5! + ... R = ∞
ln(1+x) = ∑(n=1..∞) (−1)ⁿ⁺¹ · xⁿ/n = x − x²/2 + x³/3 − ... R = 1, включая x = 1
1/(1−x) = ∑(n=0..∞) xⁿ = 1 + x + x² + x³ + ... R = 1
(1+x)ᵅ = ∑(n=0..∞) C(α,n)·xⁿ = 1 + αx + α(α−1)x²/2! + ... R = 1

где C(α,n) = α(α−1)...(α−n+1)/n! -- обобщённые биномиальные коэффициенты.

arctg x = ∑(n=0..∞) (−1)ⁿ · x²ⁿ⁺¹/(2n+1) = x − x³/3 + x⁵/5 − ... R = 1, включая x = ±1

Вывод разложения ex

f(x) = ex → f(n)(x) = ex → f(n)(0) = 1 для всех n.

Ряд Маклорена: ex = ∑ xn/n!

Проверка сходимости к f: |f(n+1)(ξ)| = eξ ≤ e|x|. Тогда |Rn| ≤ e|x|·|x|n+1/(n+1)! → 0 для любого x.

Вывод разложения sin x

f(n)(0): последовательность 0, 1, 0, −1, 0, 1, ... (цикл с периодом 4). Ненулевые только при нечётных степенях.

|f(n)(x)| ≤ 1 для всех n и x → по Теореме 6.5 ряд сходится к sin x при всех x.

Вывод разложения ln(1+x)

Начинаем с 1/(1+x) = ∑(−1)nxn (|x| < 1). Интегрируем почленно:

∫₀ˣ dt/(1+t) = ln(1+x) = ∑(−1)ⁿ · xⁿ⁺¹/(n+1) = ∑(−1)ⁿ⁺¹ · xⁿ/n

При x = 1 ряд сходится (по Лейбницу), и по теореме Абеля ln 2 = 1 − 1/2 + 1/3 − ...

Из базовых разложений можно получать новые подстановкой, дифференцированием, интегрированием. Например: e−x² = ∑(−1)nx2n/n! (подставили −x² вместо x в ex).

Применение рядов Тейлора для приближённых вычислений

Идея метода

Заменяем функцию частичной суммой ряда Тейлора: f(x) ≈ Sn(x). Остаточный член Rn(x) даёт оценку погрешности.

Оценка погрешности

  • Для знакочередующихся рядов (Лейбниц): |Rn| ≤ |an+1| -- первый отброшенный член
  • Для знакоположительных: остаток Лагранжа |Rn| ≤ M·|x|n+1/(n+1)!

Вычисление значений функций

Пример: вычислить e с точностью 0.001

e = e1 = ∑ 1/n! = 1 + 1 + 1/2! + 1/3! + ...

Ряд знакоположительный. Используем оценку: Rn < e·1n+1/(n+1)! ≤ 3/(n+1)!.

3/7! = 3/5040 ≈ 0.0006 < 0.001 → достаточно 7 членов (n=0..6).

e ≈ 1 + 1 + 0.5 + 0.1667 + 0.0417 + 0.0083 + 0.0014 = 2.7181

Вычисление интегралов, не берущихся аналитически

Пример: ∫₀¹ e^(−x²) dx

Первообразная не выражается через элементарные функции. Разлагаем:

e^(−x²) = 1 − x² + x⁴/2! − x⁶/3! + x⁸/4! − ...

Интегрируем почленно (равномерная сходимость на [0,1]):

∫₀¹ = [x − x³/3 + x⁵/(5·2!) − x⁷/(7·3!) + ...]₀¹

= 1 − 1/3 + 1/10 − 1/42 + 1/216 − ...

Знакочередующийся ряд. С точностью 0.01: |a₄| = 1/216 ≈ 0.005 < 0.01.

∫₀¹ e^(−x²)dx ≈ 1 − 0.333 + 0.1 − 0.024 = 0.743 (точное ≈ 0.7468)

Вычисление пределов

Пример

lim(x→0) (sin x − x)/x³

sin x = x − x³/6 + x⁵/120 − ...

(sin x − x)/x³ = (−x³/6 + x⁵/120 − ...)/x³ = −1/6 + x²/120 − ... → −1/6

Ряды Тейлора -- мощный инструмент для: (1) вычисления значений, (2) вычисления интегралов, (3) нахождения пределов, (4) решения дифференциальных уравнений.
Ряды Фурье (Лекция 13)

Тригонометрические ряды. Ортогональность. Коэффициенты Фурье

Тригонометрический ряд

a₀/2 + ∑(n=1..∞) (aₙ·cos(nx) + bₙ·sin(nx))

Множитель 1/2 при a₀ -- для единообразия формул коэффициентов.

Ортогональная тригонометрическая система

Система функций {1, cos x, sin x, cos 2x, sin 2x, cos 3x, sin 3x, ...} ортогональна на [−π, π].

Это означает, что интеграл произведения любых двух различных функций системы равен нулю:

∫₋π^π cos(nx)·cos(mx)dx = 0 при n ≠ m ∫₋π^π sin(nx)·sin(mx)dx = 0 при n ≠ m ∫₋π^π cos(nx)·sin(mx)dx = 0 для ВСЕХ n, m (в т.ч. n = m)

Нормы (квадраты «длин»)

∫₋π^π 1² dx = 2π ∫₋π^π cos²(nx) dx = π (n ≥ 1) ∫₋π^π sin²(nx) dx = π (n ≥ 1)

Доказательство: используем формулы cos²α = (1+cos 2α)/2 и sin²α = (1−cos 2α)/2.

Коэффициенты Фурье (вывод)

Пусть f(x) = a₀/2 + ∑(ancos nx + bnsin nx). Чтобы найти am, умножаем обе части на cos(mx) и интегрируем по [−π, π]:

∫₋π^π f(x)·cos(mx)dx = aₘ · ∫₋π^π cos²(mx)dx = aₘ · π

Все остальные слагаемые обнулились благодаря ортогональности!

a₀ = 1/π · ∫₋π^π f(x) dx aₙ = 1/π · ∫₋π^π f(x)·cos(nx) dx (n = 1, 2, 3, ...) bₙ = 1/π · ∫₋π^π f(x)·sin(nx) dx (n = 1, 2, 3, ...)
Формулы получены при предположении, что ряд можно интегрировать почленно. Это предположение -- формальный вывод. Строгое обоснование даёт теорема Дирихле.

Ряды Фурье. Условия Дирихле. Теорема Дирихле

Ряд Фурье функции f(x)

Для 2π-периодической функции f(x) записываем ряд с коэффициентами Фурье:

f(x) ~ a₀/2 + ∑(n=1..∞) (aₙ·cos(nx) + bₙ·sin(nx))

Знак ~ означает «соответствует» (формально сопоставляем), а не «равно». Равенство нужно доказать отдельно.

Условия Дирихле

Функция f(x) удовлетворяет условиям Дирихле на [−π, π], если:

  1. f(x) ограничена и абсолютно интегрируема на [−π, π]
  2. f(x) имеет конечное число точек разрыва 1-го рода (существуют конечные левый и правый пределы)
  3. f(x) имеет конечное число экстремумов (кусочно-монотонна)

Кратко: кусочно-непрерывная и кусочно-монотонная функция.

Теорема Дирихле (о поточечной сходимости ряда Фурье)

Если f(x) удовлетворяет условиям Дирихле на [−π, π], то её ряд Фурье сходится в каждой точке:

В точках непрерывности: ряд Фурье = f(x) В точках разрыва: ряд Фурье = (f(x−0) + f(x+0)) / 2

В точках разрыва сумма ряда равна среднему арифметическому левого и правого пределов.

На концах периода

Для 2π-периодической функции на концах интервала (x = ±π):

S(±π) = (f(−π + 0) + f(π − 0)) / 2

Это среднее между значением «слева от π» и «справа от −π» (периодическое продолжение).

Пример: f(x) = |x| на [−π, π]

Функция непрерывна, кусочно-монотонна → условия Дирихле выполнены.

Чётная функция → bn = 0.

a₀ = 2/π · ∫₀π x dx = 2/π · π²/2 = π

an = 2/π · ∫₀π x·cos(nx) dx = 2/(πn²) · (cos(nπ) − 1) = 2/(πn²) · ((−1)n − 1)

При чётном n: an = 0. При нечётном n: an = −4/(πn²).

|x| = π/2 − (4/π)·(cos x + cos 3x/9 + cos 5x/25 + ...)

При x = 0: 0 = π/2 − (4/π)·(1 + 1/9 + 1/25 + ...) → π²/8 = 1 + 1/9 + 1/25 + ...

Ряды Фурье чётных/нечётных функций. Произвольный промежуток [-l, l]

Чётная функция: f(−x) = f(x)

Ряд Фурье содержит только косинусы (bn = 0):

f(x) = a₀/2 + ∑(n=1..∞) aₙ·cos(nx)

Коэффициенты (интегрируем только по [0, π], удвоив):

a₀ = 2/π · ∫₀^π f(x) dx aₙ = 2/π · ∫₀^π f(x)·cos(nx) dx bₙ = 0

Нечётная функция: f(−x) = −f(x)

Ряд Фурье содержит только синусы (a₀ = 0, an = 0):

f(x) = ∑(n=1..∞) bₙ·sin(nx)

Коэффициенты:

aₙ = 0 bₙ = 2/π · ∫₀^π f(x)·sin(nx) dx

Почему так работает

cos(nx) -- чётная функция, sin(nx) -- нечётная. Произведение:

  • чётная × нечётная = нечётная → ∫−ππ = 0 → bn = 0 для чётной f
  • нечётная × чётная = нечётная → ∫−ππ = 0 → an = 0 для нечётной f
  • чётная × чётная = чётная → ∫−ππ = 2·∫0π (вот откуда множитель 2)

Разложение на полупериоде [0, π]

Если f(x) задана только на [0, π], можно продолжить её на [−π, 0]:

  • Чётное продолжение: f(−x) = f(x) → получаем ряд по косинусам
  • Нечётное продолжение: f(−x) = −f(x) → получаем ряд по синусам
Пример: f(x) = x на [−π, π]

f(x) = x -- нечётная → только синусы, an = 0.

bn = 2/π · ∫₀π x·sin(nx)dx

Интегрируем по частям: = 2/π · [−x·cos(nx)/n + sin(nx)/n²]₀π

= 2/π · (−π·cos(nπ)/n) = −2·cos(nπ)/n = 2·(−1)n+1/n

x = 2·(sin x − sin 2x/2 + sin 3x/3 − sin 4x/4 + ...)

Произвольный промежуток [−l, l]

Если функция имеет период 2l (а не 2π), делаем замену t = πx/l:

f(x) = a₀/2 + ∑(n=1..∞) (aₙ·cos(nπx/l) + bₙ·sin(nπx/l))

Коэффициенты Фурье на [−l, l]:

a₀ = 1/l · ∫₋ₗˡ f(x) dx aₙ = 1/l · ∫₋ₗˡ f(x)·cos(nπx/l) dx bₙ = 1/l · ∫₋ₗˡ f(x)·sin(nπx/l) dx

Мнемоника: везде заменяем nx → nπx/l и 1/π → 1/l.

Теорема Дирихле на [−l, l]

Формулировка та же: в точках непрерывности ряд Фурье = f(x), в точках разрыва = (f(x−0)+f(x+0))/2.

Пример: f(x) = x на [−1, 1], l = 1

f(x) = x -- нечётная → an = 0, только синусы.

bn = 1/1 · ∫−11 x·sin(nπx) dx = 2 · ∫01 x·sin(nπx) dx

= 2·[−x·cos(nπx)/(nπ) + sin(nπx)/(n²π²)]₀1 = −2·cos(nπ)/(nπ) = 2·(−1)n+1/(nπ)

x = (2/π)·∑(−1)ⁿ⁺¹·sin(nπx)/n

Определи чётность функции первым делом -- это сразу убирает половину коэффициентов и упрощает вычисления вдвое! На [−l,l] формулы аналогичны: множитель 2/l вместо 2/π, аргумент nπx/l вместо nx.
