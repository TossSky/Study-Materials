Практика 9

Логика предикатов • Множества истинности • Кванторные операции

Теория

Предикат \(P(x)\) — функция, определённая на множестве \(\mathbf{M}\), принимающая значения \(\{0, 1\}\).
Множество истинности: \[I_P = \{x \in \mathbf{M} \mid P(x) = 1\}\]
Операции и их множества истинности:
\(\overline{P}(x) = \neg P(x)\)\(I = \mathbf{M} \setminus I_P\)
\(P(x) \wedge Q(x)\)\(I = I_P \cap I_Q\)
\(P(x) \vee Q(x)\)\(I = I_P \cup I_Q\)
\(P(x) \to Q(x) = \neg P \vee Q\)\(I = \overline{I_P} \cup I_Q\)
\(P(x) \leftrightarrow Q(x)\)\(I = (I_P \cap I_Q) \cup (\overline{I_P} \cap \overline{I_Q})\)
Кванторы (на конечном \(\mathbf{M} = \{x_1, \ldots, x_N\}\)): \[\forall x\, P(x) = P(x_1) \wedge P(x_2) \wedge \ldots \wedge P(x_N)\] \[\exists x\, P(x) = P(x_1) \vee P(x_2) \vee \ldots \vee P(x_N)\]

Задание 1 — Область истинности (диаграммы Венна)

1
\(P(x)\,Q(x)\,\overline{R(x)}\)
Раскрываем
\(P \wedge Q \wedge \neg R\)
Ответ
\[I = I_P \cap I_Q \cap \overline{I_R}\] На диаграмме: пересечение \(P\) и \(Q\), но вне \(R\).
2
\(\overline{P(x)} \leftrightarrow \overline{Q(x)}\)
Упрощаем
\(\neg P \leftrightarrow \neg Q\). Эквивалентность отрицаний = эквивалентность самих предикатов:
\(\neg P \leftrightarrow \neg Q \;\equiv\; P \leftrightarrow Q\)
Ответ
\[I = (I_P \cap I_Q) \cup (\overline{I_P} \cap \overline{I_Q})\] На диаграмме: \(x\) одновременно в \(P\) и \(Q\), либо одновременно вне обоих.
3
\(P(x) \to \bigl(Q(x) \vee \overline{Q(x)}\bigr)\)
Упрощаем
\(Q \vee \neg Q = 1\) — тавтология (закон исключённого третьего).
\(P \to 1 = 1\) — импликация с истинным следствием всегда истинна.
Ответ
\[I = \mathbf{M}\] Тождественно истинна на всём множестве.
4
\(\overline{P(x)} \vee Q(x)\)
Упрощаем
\(\neg P \vee Q \;\equiv\; P \to Q\) — это определение импликации.
Ответ
\[I = \overline{I_P} \cup I_Q\] На диаграмме: всё вне \(P\), плюс всё в \(Q\).
5
\(P(x)\,Q(x) \to \overline{R(x)}\)
Раскрываем
\((P \wedge Q) \to \neg R = \neg(P \wedge Q) \vee \neg R = \neg P \vee \neg Q \vee \neg R\)
Ответ
\[I = \overline{I_P} \cup \overline{I_Q} \cup \overline{I_R} = \mathbf{M} \setminus (I_P \cap I_Q \cap I_R)\] На диаграмме: всё, кроме пересечения всех трёх кругов.
6
\(P(x)\,Q(x) \vee P(x)\,R(x)\)
Раскрываем
\((P \wedge Q) \vee (P \wedge R) = P \wedge (Q \vee R)\) — вынесли \(P\) за скобку (дистрибутивность).
Ответ
\[I = I_P \cap (I_Q \cup I_R)\] На диаграмме: часть \(P\), попадающая в \(Q\) или \(R\) (или оба).

Задание 2 — Область истинности на \(\mathbb{R}\)

1
\(P(x) = \left(\dfrac{x^2+3x+2}{x^2+4x+3} = 0\right)\)
Числитель = 0
\(x^2+3x+2 = (x+1)(x+2) = 0 \;\Rightarrow\; x = -1 \text{ или } x = -2\)
Знаменатель = 0 (запрещённые значения)
\(x^2+4x+3 = (x+1)(x+3) = 0 \;\Rightarrow\; x = -1 \text{ или } x = -3\)
Проверка корней
Дробь = 0 когда числитель = 0 и знаменатель ≠ 0:

• \(x = -1\): знаменатель = \((-1+1)(-1+3) = 0\) → не определена ✗
• \(x = -2\): знаменатель = \((-2+1)(-2+3) = (-1)(1) = -1 \neq 0\) → дробь = 0 ✓
Ответ
\[I_P = \{-2\}\]
2
\(P(x) = (x^2-6x+8 \geq 0)\;\&\;(x^2-7x+10 \leq 0)\)
Первое неравенство
\(x^2 - 6x + 8 = (x-2)(x-4) \geq 0\)
Корни: \(2, 4\). Парабола ветвями вверх.
\(\Rightarrow x \in (-\infty,\, 2] \cup [4,\, +\infty)\)
Второе неравенство
\(x^2 - 7x + 10 = (x-2)(x-5) \leq 0\)
Корни: \(2, 5\). Парабола ветвями вверх.
\(\Rightarrow x \in [2,\, 5]\)
Пересечение
\(\bigl((-\infty, 2] \cup [4, +\infty)\bigr) \;\cap\; [2, 5]\)
\(= \{2\} \cup [4, 5]\)
Ответ
\[I_P = \{2\} \cup [4,\, 5]\]
3
\(P(x, y) = (\lg(x) = \lg(y))\)
ОДЗ
\(\lg\) определён только при положительных аргументах: \(x > 0,\; y > 0\).
Решение
\(\lg(x) = \lg(y) \;\Longleftrightarrow\; x = y\) (логарифм — строго монотонная, взаимно однозначная функция).
Ответ
\[I_P = \{(x, y) \in \mathbb{R}^2 \mid x = y,\; x > 0\}\] Геометрически: луч \(y = x\) при \(x > 0\) (биссектриса первого квадранта, без начала координат).

Задание 3 — Формулы с кванторами

Даны предикаты: \(S(x,y,z) = (x+y=z)\), \;\(P(x,y,z) = (xy=z)\).
Свободная переменная — \(x\), остальные связаны кванторами. Область — \(\mathbb{Z}\).

1
\(x\) — чётное число
Идея
\(x\) чётное \(\Leftrightarrow\) \(\exists y: x = 2y\) \(\Leftrightarrow\) \(\exists y: y + y = x\)
Формула
\[\exists y\; S(y, y, x)\]
Проверка: \(S(y,y,x)\) означает \(y+y=x\), т.е. \(x = 2y\) — определение чётности. ✓
2
\(x\) — нечётное число
Идея
Нечётное = НЕ чётное = не существует \(y\), что \(y+y=x\).
Формула
\[\neg\,\exists y\; S(y, y, x) \;\;\equiv\;\; \forall y\; \neg\,S(y, y, x)\]
Проверка: для любого \(y\) выполняется \(y+y \neq x\) → \(x\) не представимо как \(2y\). ✓
3
\(x = 0\)
Идея
\(0\) — нейтральный элемент сложения: \(\forall y:\; x + y = y\).
Формула
\[\forall y\; S(x, y, y)\]
Проверка: \(S(x,y,y)\) означает \(x+y=y\) → \(x=0\). Обратно: если \(x=0\), то \(0+y=y\) для любого \(y\). ✓
4
\(x = 1\)
Идея
\(1\) — нейтральный элемент умножения: \(\forall y:\; x \cdot y = y\).
Формула
\[\forall y\; P(x, y, y)\]
Проверка: \(P(x,y,y)\) означает \(xy = y\). При \(y \neq 0\): \(x = 1\). При \(y = 0\): \(x \cdot 0 = 0\) верно для любого \(x\), но формула требует выполнения для всех \(y\), включая ненулевые → \(x = 1\). ✓
5
\(x = 2\)
Идея
\(x = 2 = 1 + 1\). Сначала находим «\(1\)» через его свойство (нейтральный по умножению), затем складываем его с собой.
Формула
\[\exists y\;\bigl(\forall z\; P(y, z, z) \;\wedge\; S(y, y, x)\bigr)\]
Проверка:
• \(\forall z\; P(y,z,z)\) означает \(yz = z\) для всех \(z\) → \(y = 1\)
• \(S(y,y,x)\) означает \(y + y = x\) → \(1 + 1 = x\) → \(x = 2\)
Итого: существует \(y = 1\), для которого оба условия выполнены ⇔ \(x = 2\). ✓

Задание 4 — Формализация рассуждений

1

«Все политики — лицедеи. Некоторые лицедеи — лицемеры. Значит, некоторые политики — лицемеры.»

Вводим предикаты
\(A(x)\) — «\(x\) политик», \;\(B(x)\) — «\(x\) лицедей», \;\(C(x)\) — «\(x\) лицемер»
Посылки
1. \(\forall x\;\bigl(A(x) \to B(x)\bigr)\) — все политики — лицедеи
2. \(\exists x\;\bigl(B(x) \wedge C(x)\bigr)\) — некоторые лицедеи — лицемеры
Заключение
\(\exists x\;\bigl(A(x) \wedge C(x)\bigr)\) — некоторые политики — лицемеры
Формула рассуждения
\[\bigl(\forall x(A(x) \to B(x)) \;\wedge\; \exists x(B(x) \wedge C(x))\bigr) \;\to\; \exists x(A(x) \wedge C(x))\]
Замечание: это рассуждение НЕ является общезначимым — «некоторые лицедеи-лицемеры» могут не быть политиками. Задание просит записать, а не доказать.
2

«Глупец был бы способен на это. Я на это не способен. Значит, я не глупец.»

Вводим предикаты
\(G(x)\) — «\(x\) глупец», \;\(S(x)\) — «\(x\) способен на это», \;\(a\) — «я» (константа)
Посылки
1. \(\forall x\;\bigl(G(x) \to S(x)\bigr)\) — каждый глупец способен
2. \(\neg S(a)\) — я не способен
Формула рассуждения
\[\bigl(\forall x(G(x) \to S(x)) \;\wedge\; \neg S(a)\bigr) \;\to\; \neg G(a)\]
Проверка: это modus tollens — общезначимое рассуждение. Если \(G(a)\) было бы истинно, то по первой посылке \(S(a)\) — противоречие со второй посылкой. ✓
3

«Друг моего друга — мой друг.»

Вводим предикаты
\(F(x, y)\) — «\(x\) друг \(y\)», \;\(a\) — «я» (константа)
Формула
\[\forall x\;\forall y\;\bigl(F(x, a) \wedge F(y, x) \;\to\; F(y, a)\bigr)\]
Чтение: для любых \(x, y\): если \(x\) — мой друг и \(y\) — друг \(x\), то \(y\) — мой друг. ✓
4

«Каждый любит сам себя. Значит, кого-то кто-нибудь любит.»

Вводим предикаты
\(L(x, y)\) — «\(x\) любит \(y\)»
Посылка и заключение
Посылка: \(\forall x\; L(x, x)\) — каждый любит себя
Заключение: \(\exists y\;\exists x\; L(x, y)\) — существует кто-то, кого кто-нибудь любит
Формула рассуждения
\[\forall x\; L(x, x) \;\to\; \exists y\;\exists x\; L(x, y)\]
Проверка: общезначимо. Если каждый любит себя, возьмём любого человека \(y\) — он сам себя любит, т.е. \(\exists x\; L(x, y)\) (этот \(x = y\)). ✓
