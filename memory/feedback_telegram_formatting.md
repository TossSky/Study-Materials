При ответах в Telegram использовать markdownv2 с разделительными чертами, жирным, курсивом, моноширным шрифтом.

**Why:** Длинные ответы без форматирования плохо читаются на телефоне.
**How to apply:** Использовать format: markdownv2 и экранировать спецсимволы MarkdownV2 (`.`, `!`, `-`, `(`, `)`, `>`, `#`, `+`, `=`, `{`, `}`, `|`, `~`).
