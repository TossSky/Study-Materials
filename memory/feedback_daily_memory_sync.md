Каждый день обновлять файлы памяти и прогресса.

**Why:** Пользователь работает с разных устройств и в разных сессиях. Если память устареет — контекст теряется и приходится заново объяснять.

**How to apply:**
1. В конце каждого рабочего дня (или при значимых изменениях) обновлять `project_semester4_progress.md` — статусы задач, новые дедлайны, что сделано
2. Синхронизировать память из `.claude/projects/.../memory/` → `Study-Materials/memory/` (локальная копия в гите)
3. Обновлять CLAUDE.md если появилась важная новая информация
4. Не забывать обновлять дату "Последнее обновление" в файле прогресса
