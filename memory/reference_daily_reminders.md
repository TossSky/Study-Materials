# Ежедневные напоминания

Когда Вячеслав просит "включи напоминания" или "напоминай мне каждый день" — настрой два CronCreate:

## 1. Утреннее (10:00 МСК)
**Cron:** `57 9 * * *`
**Что делать:**
- Прочитай `memory/project_semester4_progress.md` — актуальный статус
- Посмотри какой сегодня день недели (влияет на расписание пар)
- Отправь в Telegram (chat_id: 916411940):
  1. Какой день, сколько дней до сессии (25.05.2026)
  2. Приоритетные задачи на сегодня (1-2 штуки)
  3. Краткая мотивация

## 2. Вечернее (20:00 МСК)
**Cron:** `3 20 * * *`
**Что делать:**
- Прочитай `memory/project_semester4_progress.md`
- Отправь в Telegram:
  1. План на завтра (1-2 задачи)
  2. Напомнить про пары если есть
  3. Краткий итог дня (если есть информация)

## Приоритеты задач
1. Ближайшие дедлайны (АСВТ зачёт до 27.04.2026)
2. Экзамены сессии (с 25.05.2026)
3. Академ разница (ОИБ, История)
4. Английский — НЕ в приоритете
